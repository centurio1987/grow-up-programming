알고리즘 — 코딩 테스트·실무 활용도 순 정리
"코딩 테스트 빈출"과 "실무 활용"을 함께 고려해 상(필수) → 중(빈출) → 하(특정 분야/고급) 3단계로 묶었습니다. 각 단계 내에서도 위에 있을수록 더 자주 등장합니다.
★★★ 상 — 필수 (코딩 테스트 단골 + 실무 일상)
거의 모든 코딩 테스트 합격선이 여기에서 갈리고, 실무에서도 자료구조·데이터베이스·검색·캐싱 등에 직접 쓰입니다.

이진 탐색 (Binary Search) — 정렬된 데이터 탐색, "답을 이분 탐색"하는 파라메트릭 서치 (O(log n))
해시맵 / 해시셋 — 빈도 카운트, 중복 제거, 두 수의 합류 문제 (평균 O(1))
두 포인터 (Two Pointers) — 정렬된 배열에서 쌍 찾기, 부분합 (O(n))
슬라이딩 윈도우 — 연속 부분 배열·문자열의 최댓·최솟·길이 (O(n))
누적합 (Prefix Sum) — 구간 합 질의, 부분합이 K인 구간 개수 (전처리 O(n), 질의 O(1))
정렬 — Quicksort/Mergesort/Timsort, 실무에선 언어 내장 정렬 활용 (O(n log n))
BFS — 무가중치 최단 경로, 레벨 탐색, 미로 문제 (O(V+E))
DFS / 백트래킹 — 연결 성분, 경로 탐색, 조합·순열 생성 (O(V+E))
스택·큐·덱 — 괄호 검증, 단조 스택, BFS 보조 자료구조
Kadane's Algorithm (최대 부분합) — 1D DP 입문, 면접 단골 (O(n))
1차원 DP — 계단 오르기, House Robber, 동전 교환, 0/1 배낭 (O(nW))
그리디 — 회의실 배정, 거스름돈, 최소 동전 (정렬 + 일관 선택)
재귀·분할 정복 — 트리 순회, 머지 정렬, 이진 탐색의 기반
Union-Find (Disjoint Set Union) — 연결성·MST·집합 병합 (거의 O(1), α(n))
트리 순회 (전위·중위·후위·레벨) — DFS/BFS의 트리 버전, 면접 빈출
연결 리스트 기본 연산 — 뒤집기, 사이클 검출(Floyd), 두 리스트 병합

★★ 중 — 빈출 (코딩 테스트 중상위 + 실무에서 종종)
상위권 코딩 테스트(삼성 SW 역량 테스트, 카카오 블라인드, 구글 인터뷰 등)와 백엔드·데이터 엔지니어링 실무에서 자주 등장합니다.

Dijkstra 최단 경로 — 가중치 그래프, 지도·네트워크 라우팅 (O((V+E) log V))
위상 정렬 — 빌드 시스템, 작업 스케줄링, 강의 수강 순서 (O(V+E))
2D DP — LCS, 편집 거리, 격자 경로 (O(nm))
LIS (최장 증가 부분 수열) — DP+이분 탐색 (O(n log n))
최소 신장 트리 (Kruskal·Prim) — 네트워크 설계, 클러스터링 (O(E log E))
세그먼트 트리 / Fenwick Tree (BIT) — 동적 구간 질의, 실시간 랭킹·집계 (O(log n))
Trie — 자동완성, 사전, IP 라우팅 테이블, 검색 엔진
문자열 검색 (KMP, Z, Rabin-Karp) — grep, 보안 패턴 매칭, DNA 검색 (O(n+m))
힙 / 우선순위 큐 — 상위 K개, 스트림 중앙값, Dijkstra·이벤트 시뮬레이션
Top-K 문제 — Min-Heap (O(n log k)) 또는 Bucket Sort
Quickselect (k번째 원소) — 중앙값, 백분위수 (평균 O(n))
비트마스크 — 부분집합 열거, 작은 N의 상태 DP, 권한 플래그
회전된 정렬 배열 탐색 — 변형 이분 탐색 (면접 단골)
단조 스택 (Monotonic Stack) — Next Greater Element, 히스토그램 최대 직사각형
단조 덱 (Monotonic Deque) — 슬라이딩 윈도우 최댓값
사이클 검출 (유향·무향) — DAG 판정, 데드락 검출
이분 그래프 판정 (2색칠) — 충돌 그래프, 매칭 가능성 사전 체크
부분집합 합·0/1 배낭 변형 — 선택 최적화 문제 전반
트리 DP — 서브트리 집계, 독립집합, 회사 조직도 류 문제
구간 DP (행렬 체인, 팰린드롬 분할) — 분할 최적화 (O(n³))
자리수 DP (Digit DP) — "1~N 중 조건 만족 수" 카운팅
거듭제곱 (Fast Exponentiation) — 모듈러 거듭제곱, RSA, 행렬 거듭제곱 (O(log n))
GCD / 확장 유클리드 — 모듈러 역원, 기약분수, 디오판토스
에라토스테네스의 체 — 소수 목록, 정수론 문제 전반 (O(n log log n))
LRU 캐시 구현 — 해시맵 + 이중 연결 리스트, 시스템 디자인 면접 단골
레이트 리미터 / 토큰 버킷 — 큐 + 시간 기반 (실무 API 게이트웨이)

★ 하 — 특정 분야·고급 (상위 대회·전문 도메인)
알고리즘 대회(ICPC, Codeforces), 보안·검색·그래픽스·생물정보학 같은 특정 실무 도메인에서 등장합니다.
그래프·네트워크 고급

Bellman-Ford / SPFA — 음수 간선, 차익 거래·환율 탐지 (O(VE))
Floyd-Warshall — 모든 쌍 최단 경로, 작은 그래프 (O(V³))
0-1 BFS — 가중치 0/1 그래프의 최단 경로 (O(V+E))
A\* 탐색 — 게임 AI 길찾기, 휴리스틱 기반 Dijkstra
SCC (Tarjan·Kosaraju) — 2-SAT, 의존성 분석
단절점·다리 (Tarjan) — 네트워크 취약점 분석
최대 유량 (Edmonds-Karp, Dinic) — 매칭·할당 일반화, 이미지 세분화
이분 매칭 (Hopcroft-Karp, Hungarian) — 작업 배정, 광고 매칭
최소 비용 최대 유량 (MCMF) — 물류·트래픽 최적화

문자열 고급

자료구조 고급

수학·정수론·기하
