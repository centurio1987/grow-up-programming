알고리즘 문제 — 표준 해법 쌍 목록
분야별로 정리했습니다. 각 항목은 문제 유형 → 대표 알고리즘 (핵심 아이디어, 시간복잡도) 형식입니다.

1. 배열·구간 문제 (DP 1차원 패턴)

Maximum Slice (최대 부분합) → Kadane's Algorithm (현재 위치에서 끝나는 최대합 점화, O(n))
최대 곱 부분 배열 → Kadane 변형 (음수 곱셈 때문에 최댓값·최솟값 동시 추적, O(n))
주식 매매 최대 이익 (1회) → 좌측 최솟값 추적 (각 시점까지의 최저가 갱신, O(n))
주식 매매 최대 이익 (k회) → DP (상태: 거래 횟수·보유 여부, O(nk))
House Robber (인접 불가 최대합) → DP (dp[i] = max(dp[i-1], dp[i-2]+a[i]), O(n))
최장 증가 부분 수열 (LIS) → DP+이분 탐색 (patience sorting, O(n log n))
최장 공통 부분 수열 (LCS) → 2D DP (O(nm))
편집 거리 (Levenshtein) → 2D DP (삽입/삭제/교체, O(nm))
부분합이 K인 부분배열 개수 → 누적합 + 해시맵 (prefix[j] - prefix[i] = k, O(n))
구간 합 질의 (정적) → Prefix Sum (전처리 O(n), 질의 O(1))
구간 합 질의 (동적 갱신) → Fenwick Tree (BIT) 또는 Segment Tree (갱신·질의 O(log n))
구간 최솟값 질의 (정적) → Sparse Table (전처리 O(n log n), 질의 O(1))
구간 최솟값 질의 (동적) → Segment Tree (O(log n))
구간 갱신 + 점 질의 → 차분 배열 또는 Lazy Segment Tree
슬라이딩 윈도우 최댓값 → Monotonic Deque (O(n))
가장 가까운 큰 수 (NGE) → Monotonic Stack (O(n))
히스토그램 최대 직사각형 → Monotonic Stack (O(n))
연속 부분배열 합 ≤ S 최장 길이 → Two Pointers / Sliding Window (O(n))

2. 정렬·선택

일반 정렬 → Quicksort (평균 O(n log n)), Mergesort (안정, O(n log n)), Heapsort (제자리, O(n log n))
거의 정렬된 배열 → Insertion Sort (O(n + 역순쌍 수))
작은 정수 키 정렬 → Counting Sort (O(n+k))
고정 길이 문자열·정수 정렬 → Radix Sort (O(d(n+k)))
외부 정렬 (메모리 초과) → External Merge Sort
k번째 작은 원소 → Quickselect (평균 O(n)), Median of Medians (최악 O(n))
스트림에서 중앙값 → 두 개의 힙 (max-heap + min-heap, O(log n) 삽입)
상위 K개 빈도 → Min-Heap (O(n log k)) 또는 Bucket Sort (O(n))

3. 탐색

정렬된 배열 탐색 → Binary Search (O(log n))
단조 함수의 경계 찾기 → Parametric Binary Search / "답을 이분 탐색" (O(log(범위) × 판정))
회전된 정렬 배열 탐색 → 변형 이분 탐색 (O(log n))
실수 영역 최적화 → Ternary Search (단봉 함수, O(log(1/ε)))

4. 그래프 — 탐색·연결성

연결 성분 / 도달 가능성 → BFS / DFS (O(V+E))
무가중치 최단 경로 → BFS (O(V+E))
0-1 가중치 최단 경로 → 0-1 BFS (Deque, O(V+E))
사이클 탐지 (무향) → Union-Find 또는 DFS
사이클 탐지 (유향) → DFS 색칠 (white/gray/black)
위상 정렬 → Kahn's Algorithm (BFS, 진입차수) 또는 DFS 후위 순회 (O(V+E))
강한 연결 요소 (SCC) → Tarjan's 또는 Kosaraju's (O(V+E))
이중 연결 요소 / 단절점·다리 → Tarjan's Algorithm (low-link, O(V+E))
다리(Bridge) 검출 → Tarjan's Bridge-finding (O(V+E))

5. 그래프 — 최단 경로

단일 시작점, 음수 없음 → Dijkstra (이진 힙 O((V+E) log V), 피보나치 힙 O(E + V log V))
단일 시작점, 음수 가능 → Bellman-Ford (O(VE), 음수 사이클 검출 가능)
음수 간선, 희소 그래프 → SPFA (Bellman-Ford 큐 최적화, 평균 빠름·최악 O(VE))
모든 쌍 최단 경로 → Floyd-Warshall (O(V³))
DAG 최단 경로 → 위상 정렬 후 완화 (O(V+E))
A 탐색\* → Dijkstra + 휴리스틱 (목표 지향 탐색)

6. 그래프 — 신장 트리·매칭·플로우

최소 신장 트리 (희소) → Kruskal + Union-Find (O(E log E))
최소 신장 트리 (밀집) → Prim + 이진 힙 (O((V+E) log V))
이분 그래프 판정 → BFS 2색칠 (O(V+E))
이분 매칭 (최대) → Hopcroft-Karp (O(E√V)) 또는 Hungarian (가중, O(V³))
최대 유량 → Edmonds-Karp (O(VE²)), Dinic's (O(V²E), 단위 용량은 O(E√V))
최소 컷 → 최대 유량 = 최소 컷 정리 적용
최소 비용 최대 유량 → SSP (Successive Shortest Path) with Bellman-Ford/Dijkstra+potential

7. 트리

트리 지름 → DFS 2회 또는 트리 DP (O(n))
최소 공통 조상 (LCA) → Binary Lifting (전처리 O(n log n), 질의 O(log n)), Euler Tour + RMQ (질의 O(1))
부분 트리 질의 → DFS 오일러 순서 + Fenwick/Segment Tree
경로 질의 (트리 위) → Heavy-Light Decomposition (질의 O(log² n))
트리에서 모든 노드의 답 → Rerooting Technique (O(n))
트리 동형 판정 → AHU 알고리즘 (해싱, O(n))

8. 문자열

부분 문자열 검색 → KMP (실패함수, O(n+m)), Z-Algorithm (O(n+m)), Rabin-Karp (롤링 해시, 평균 O(n+m))
다중 패턴 검색 → Aho-Corasick (트라이 + 실패 링크, O(n + 패턴길이합 + 매칭수))
최장 회문 부분 문자열 → Manacher's Algorithm (O(n))
접미사 정렬 → Suffix Array (O(n log n) 또는 SA-IS O(n))
접미사 LCP → Kasai's Algorithm (SA 위에서, O(n))
접미사 구조 통합 → Suffix Automaton (O(n)), Suffix Tree (Ukkonen O(n))
사전 / 접두사 검색 → Trie
공백 압축된 사전 → Radix Tree / Patricia Trie

9. 동적 계획법 (2차원 이상·고전)

0/1 배낭 → DP (dp[i][w], O(nW))
무한 배낭 (동전 교환) → 1차원 DP (O(nW))
동전으로 금액 만드는 경우의 수 → 1차원 DP (순서 무관: 동전 바깥 루프, O(nW))
행렬 체인 곱셈 → 구간 DP (O(n³))
팰린드롬 분할 → 구간 DP (O(n²))
외판원 문제 (TSP) → 비트마스크 DP (Held-Karp, O(n² 2ⁿ))
부분집합 합 (Subset Sum) → DP (O(nS)) 또는 비트셋 최적화
트리 DP (독립집합 등) → 후위 순회 DP (O(n))
숫자 DP (조건 만족 수 세기) → 자리수 DP (상태: 자리·tight·기타)
확률·기댓값 → DP 또는 가우스 소거법 (마르코프 체인)

10. 수학·정수론

최대공약수 → Euclidean Algorithm (O(log min(a,b)))
모듈러 역원 / 디오판토스 → Extended Euclidean
거듭제곱 → Fast Exponentiation (O(log n))
소수 판정 (작은 수) → 시행 나눗셈 (O(√n))
소수 판정 (큰 수) → Miller-Rabin (확률적), 결정적 변형
소수 목록 → Sieve of Eratosthenes (O(n log log n)), 선형 체 (O(n))
소인수분해 (큰 수) → Pollard's rho
이산 로그 → Baby-step Giant-step (O(√p))
중국인의 나머지 정리 → CRT 구성
다항식 곱셈 / 큰 수 곱셈 → FFT / NTT (O(n log n))
조합 수 (mod p) → 팩토리얼 전처리 + Fermat 역원, Lucas 정리(소수 p, 큰 n)

11. 기하

볼록 껍질 (Convex Hull) → Graham Scan 또는 Andrew's Monotone Chain (O(n log n))
두 선분 교차 판정 → CCW (외적 부호)
점이 다각형 내부인지 → Ray Casting 또는 Winding Number
가장 가까운 두 점 쌍 → 분할 정복 (O(n log n))
여러 선분의 교차 → Bentley-Ottmann 스위프 라인 (O((n+k) log n))
다각형 면적 → Shoelace Formula (O(n))
회전 캘리퍼스 (지름·최소 둘러싼 사각형) → Rotating Calipers (볼록 껍질 후 O(n))

12. 자료구조 — 고급

누적합 갱신 + 구간 합 → Fenwick Tree (BIT)
구간 최솟값·합 등 일반 질의 → Segment Tree (+ Lazy Propagation)
k번째 원소 / 순위 → Order Statistic Tree (BIT + 좌표압축, 또는 Policy Tree)
동적 연결성 → Union-Find (경로 압축 + 랭크, α(n))
오프라인 동적 연결성 → Link-Cut Tree, Euler Tour Tree
머지·분할이 잦은 집합 → Treap, Splay Tree
구간 제곱근 분해 질의 → Mo's Algorithm (오프라인, O((n+q)√n))
빈도 추정 (스트림) → Count-Min Sketch, HyperLogLog (근사)
집합 존재 여부 (메모리 절약) → Bloom Filter

13. 분할 정복·기타 패턴

역순쌍 개수 → Merge Sort 변형 (O(n log n))
최댓값·최솟값 동시 → 분할 정복 (비교 3⌈n/2⌉회)
부분합으로 분할 정복 최적화 → Divide & Conquer DP (단조성 조건, O(n² log n) → O(n²))
사면체 부등식 만족 DP → Knuth's Optimization (O(n²))
CHT 가능한 DP → Convex Hull Trick (O(n log n) 또는 O(n))
백트래킹 (N-Queens, 스도쿠) → DFS + 가지치기
Meet in the Middle → 절반씩 나눠 결합 (Subset Sum 등, O(2^(n/2)))

14. 비트·기타 트릭

부분집합 순회 → for (s = m; s; s = (s-1) & m) (서브마스크 열거)
최하위 1비트 → x & -x (Fenwick Tree 핵심 연산)
xor 누적 트릭 → 한 개만 홀수 번 나오는 원소 찾기 (O(n), 공간 O(1))
반복 제곱 / 행렬 거듭제곱 → 선형 점화식 빠르게 (피보나치 O(log n))
