# Two Sum

## 한 줄 요약

> 함수는 정수 배열과 목표값을 받아, 합이 목표값이 되는 두 원소의 인덱스를 반환한다.

## 스토리

쇼핑몰 계산대에서 직원이 정확히 목표 금액을 만들기 위해 선반에서 두 상품을 골라야 한다. 선반에는 다양한 가격의 상품들이 순서대로 놓여 있고, 두 상품의 가격 합이 정확히 목표 금액이 되어야 한다.

같은 상품을 두 번 가져갈 수는 없다. 문제에서는 합이 목표값이 되는 두 상품의 쌍이 반드시 존재한다고 보장한다.

두 상품이 선반 위 몇 번째에 있는지 인덱스 쌍으로 반환하라.

## 함수 인터페이스

```ts
export function twoSum(nums: number[], target: number): [number, number];
```

- `nums` — 길이 $N$인 정수 배열
- `target` — 두 원소의 합이 되어야 하는 목표값
- 반환 — `nums[i] + nums[j] === target`을 만족하는 인덱스 쌍 `[i, j]` ($i < j$)

## 제약 조건

- $2 \leq N \leq 10{,}000$ ($N$은 `nums`의 길이)
- `nums`의 원소는 정수 (음수 포함 가능)
- 합이 `target`이 되는 두 인덱스는 반드시 존재한다
- 같은 인덱스를 두 번 사용하지 않는다
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

배열 `nums`와 정수 `target`이 주어질 때, `nums[i] + nums[j] === target`이고 $i \neq j$인 두 인덱스 $(i, j)$를 찾아 `[i, j]` 형태로 반환한다. $i < j$가 되도록 작은 인덱스를 먼저 반환한다.

## 예시

```ts
twoSum([2, 7, 11, 15], 9);
// [0, 1] — nums[0]=2, nums[1]=7, 2+7=9

twoSum([3, 2, 4], 6);
// [1, 2] — nums[1]=2, nums[2]=4, 2+4=6

twoSum([3, 3], 6);
// [0, 1] — 같은 값이지만 서로 다른 인덱스

twoSum([-1, -2, 3], 1);
// [1, 2] — nums[1]=-2, nums[2]=3, -2+3=1

twoSum([0, 5, 0], 0);
// [0, 2] — nums[0]=0, nums[2]=0, 0+0=0

twoSum([-3, 4, -7], -10);
// [0, 2] — nums[0]=-3, nums[2]=-7, -3+(-7)=-10
```
