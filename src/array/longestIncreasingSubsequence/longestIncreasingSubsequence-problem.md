# Longest Increasing Subsequence (LIS)

## 함수 인터페이스

```ts
export function longestIncreasingSubsequence(A: number[]): number;
```

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$ 은 `A` 의 길이)
- $-10^9 \leq A[i] \leq 10^9$ (정수)

## 문제 상세

정수 배열 $A$ 가 주어진다. **엄격하게 증가하는 부분 수열** 중 가장 긴 것의 길이를 반환한다.

**부분 수열** 이란 원소 순서를 유지하되 일부 원소를 건너뛴 새 수열이며, **엄격 증가** 란 연속한 두 원소가 $a < b$ 를 만족하는 것을 의미한다 (같은 값 허용되지 않음).

즉, 인덱스 수열 $i_0 < i_1 < \cdots < i_{L-1}$ 가

$$A[i_0] < A[i_1] < \cdots < A[i_{L-1}]$$

를 만족할 때 가능한 가장 큰 $L$ 을 반환한다.

## 예시

| `A` | 반환값 | 가장 긴 LIS 예 |
| --- | --- | --- |
| `[10, 9, 2, 5, 3, 7, 101, 18]` | `4` | `[2, 3, 7, 18]` |
| `[0, 1, 0, 3, 2, 3]` | `4` | `[0, 1, 2, 3]` |
| `[7, 7, 7, 7]` | `1` | `[7]` (엄격 증가이므로) |
| `[1]` | `1` | `[1]` |
| `[5, 4, 3, 2, 1]` | `1` | `[5]` |
