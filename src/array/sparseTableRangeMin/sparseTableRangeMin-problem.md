# 구간 최솟값 질의 (정적 배열, Sparse Table)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 고급 |

## 함수 인터페이스

```ts
export function sparseTableRangeMin(
  A: number[],
  queries: Array<[number, number]>,
): number[];
```

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$ 은 `A` 의 길이)
- $1 \leq Q \leq 100{,}000$ (여기서 $Q$ 는 `queries` 의 길이)
- $-10^9 \leq A[i] \leq 10^9$ (정수)
- 각 질의 $(l, r)$ 는 $0 \leq l \leq r \leq N - 1$
- `A` 는 질의 중 갱신되지 않는다 (정적 배열)

## 문제 상세

정수 배열 $A$ 와 여러 개의 질의 $(l, r)$ 가 주어진다. 각 질의에 대해

$$\min(A[l], A[l+1], \ldots, A[r])$$

을 계산하고, 모든 질의의 결과를 입력 순서대로 담은 배열을 반환한다.

## 예시

```ts
sparseTableRangeMin(
  [3, 1, 4, 1, 5, 9, 2, 6],
  [
    [0, 7],  // 1
    [0, 2],  // 1
    [2, 2],  // 4
    [4, 7],  // 2
    [4, 5],  // 5
  ],
);
// [1, 1, 4, 2, 5]

sparseTableRangeMin([7, -3, 2, -1], [[0, 3], [1, 1], [2, 3]]);
// [-3, -3, -1]
```
