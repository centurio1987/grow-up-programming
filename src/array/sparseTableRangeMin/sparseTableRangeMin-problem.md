# 구간 최솟값 질의 (정적 배열)

## 한 줄 요약

> 정수 배열과 구간 질의 목록을 받아, **각 질의에 해당하는 구간 최솟값**을 순서대로 담은 배열을 반환한다.

## 스토리

기상청 통계 분석가 민서는 수년치 일별 기온 기록을 가지고 있다. 기록은 더 이상 변하지 않는 확정 데이터다.

동료 연구원들이 수시로 질의를 보낸다. "날짜 $l$부터 날짜 $r$ 사이에 기온이 가장 낮았던 날의 온도는?" 같은 형태다. 질의는 대량으로 들어오지만 기온 데이터는 변하지 않는다.

민서는 모든 질의에 대한 구간 최솟값을 빠르게 계산해 순서대로 반환해야 한다.

## 함수 인터페이스

```ts
export function sparseTableRangeMin(
  A: number[],
  queries: Array<[number, number]>,
): number[];
```

- `A` — 정적 정수 배열. 질의 중 변경되지 않는다.
- `queries` — 질의 목록. 각 원소 `[l, r]`은 인덱스 $[l, r]$ 구간의 최솟값을 요청한다.
- 반환 — 각 질의의 최솟값을 입력 순서대로 담은 배열.

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$은 `A`의 길이)
- $0 \leq Q \leq 100{,}000$ (여기서 $Q$는 `queries`의 길이)
- $-10^9 \leq A[i] \leq 10^9$ (정수)
- 각 질의 $[l, r]$는 $0 \leq l \leq r \leq N - 1$
- `A`는 질의 중 변경되지 않는다 (정적 배열)
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

각 질의 $[l, r]$에 대해

$$\min(A[l], A[l+1], \ldots, A[r])$$

을 계산한다. 배열 $A$는 질의 중 변하지 않는다. 질의가 없으면 빈 배열을 반환한다. $l = r$이면 단일 원소의 최솟값, 즉 `A[l]` 자체를 반환한다.

## 예시

```ts
sparseTableRangeMin(
  [3, 1, 4, 1, 5, 9, 2, 6],
  [[0, 7], [0, 2], [2, 2], [4, 7], [4, 5]],
); // [1, 1, 4, 2, 5]
// [0,7]: 전체 최솟값 1, [0,2]: min(3,1,4)=1, [2,2]: 4, [4,7]: min(5,9,2,6)=2, [4,5]: min(5,9)=5

sparseTableRangeMin([7, -3, 2, -1], [[0, 3], [1, 1], [2, 3]]);
// [-3, -3, -1]  — 전체 최솟값, 단일 인덱스, 부분 구간

sparseTableRangeMin([5, 3, 7, 1, 9], [[0, 4]]);
// [1]  — 전체 범위 최솟값

sparseTableRangeMin([1, 2, 3], []);
// []  — 질의 없음

sparseTableRangeMin([42], [[0, 0]]);
// [42]  — 단일 원소
```
