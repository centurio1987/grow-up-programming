# Largest Rectangle in Histogram

## 함수 인터페이스

```ts
export function largestRectangleInHistogram(heights: number[]): number;
```

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$ 은 `heights` 의 길이)
- $0 \leq heights[i] \leq 10^4$ (비음의 정수)

## 문제 상세

비음의 정수 배열 $heights$ 가 히스토그램 각 막대의 높이를 나타낸다. 각 막대의 너비는 $1$ 이고, 막대들은 가로 방향으로 빈틈없이 이어져 있다.

이 히스토그램에서 **막대들로 만들 수 있는 가장 큰 직사각형의 넓이** 를 반환한다.

직사각형은 어떤 연속 구간 $[l, r]$ 에 걸쳐 있는 막대들을 모두 포함해야 하며, 그 높이는 해당 구간 내 막대 높이의 최솟값으로 제한된다. 즉

$$\text{area}(l, r) = (r - l + 1) \cdot \min_{l \leq i \leq r} heights[i]$$

이고, 결과는

$$\text{maxArea}(heights) = \max_{0 \leq l \leq r < N}\, \text{area}(l, r)$$

이다.

## 예시

| `heights` | 반환값 | 설명 |
| --- | --- | --- |
| `[2, 1, 5, 6, 2, 3]` | `10` | 인덱스 $[2, 3]$ 구간, 높이 $5$, 너비 $2$ |
| `[2, 4]` | `4` | 인덱스 $[1, 1]$, 높이 $4$, 너비 $1$ |
| `[0]` | `0` | |
| `[1, 1, 1, 1]` | `4` | 너비 $4$, 높이 $1$ |
| `[6, 2, 5, 4, 5, 1, 6]` | `12` | 인덱스 $[2, 4]$, 높이 $4$, 너비 $3$ |
