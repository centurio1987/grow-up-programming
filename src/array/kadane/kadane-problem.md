# Maximum Subarray Sum

## 한 줄 요약

> 정수 배열을 받아, **연속된 부분 배열의 합 중 최댓값**을 반환한다.

## 스토리

기상 연구소의 연구원 서아는 $N$일간의 일교차 데이터를 분석한다. 각 날의 값은 양수일 수도 음수일 수도 있다.

서아는 연속된 특정 기간의 일교차 합이 최대인 구간을 찾고 싶다. 단, 구간은 반드시 하루 이상이어야 한다.

모든 날의 값이 음수라도 포기할 수 없다. 가장 덜 손해인 하루를 골라 그 값을 반환해야 한다.

## 함수 인터페이스

```ts
export function kadane(A: number[]): number;
```

- `A` — 정수 배열.
- 반환 — 비어 있지 않은 연속 부분 배열의 합 중 최댓값.

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$은 `A`의 길이)
- $-10{,}000 \leq A[i] \leq 10{,}000$ (정수, 음수 가능)
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

인덱스 구간 $[l, r]$ ($0 \leq l \leq r < N$)에 대응하는 부분 배열의 합을

$$S(l, r) = \sum_{k=l}^{r} A[k]$$

로 정의하면 답은

$$\text{answer} = \max_{0 \leq l \leq r < N}\, S(l, r)$$

이다. 빈 부분 배열($l > r$)은 허용되지 않는다. 따라서 모든 원소가 음수일 경우에도 원소 하나를 반드시 포함해야 하며, 가장 큰 단일 원소 값이 답이 된다.

## 예시

```ts
kadane([-2, 1, -3, 4, -1, 2, 1, -5, 4]); // 6  — 부분 배열 [4,-1,2,1]의 합
kadane([1, 2, 3, 4, 5]);                  // 15 — 전체가 양수이므로 전체 합
kadane([5, -3, 5]);                       // 7  — 전체 합 5+(-3)+5=7이 최대

kadane([-3, -1, -4, -2]);                 // -1 — 모두 음수, 가장 큰 단일 원소
kadane([-1, 0, -2]);                      // 0  — 단일 원소 0이 최대

kadane([7]);                              // 7  — 원소 하나
kadane([-7]);                             // -7 — 음수 원소 하나, 선택 불가피
```
