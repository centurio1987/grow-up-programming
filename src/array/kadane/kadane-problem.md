# Maximum Subarray Sum (Kadane)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★★★ 상 — 필수 |
| 난이도 | 초급 |

## 함수 인터페이스

```ts
export function kadane(A: number[]): number;
```

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$ 은 `A` 의 길이)
- $-10{,}000 \leq A[i] \leq 10{,}000$ (정수)

## 문제 상세

정수 배열 $A$ 가 주어진다. **연속된 (비어있지 않은) 부분 배열의 합 중 최댓값** 을 반환한다.

부분 배열은 인덱스 구간 $[l, r]$ ($0 \leq l \leq r < N$) 에 대응되는 $A[l..r]$ 이며, 그 합은

$$S(l, r) = \sum_{k=l}^{r} A[k]$$

이다. 결과는

$$\text{kadane}(A) = \max_{0 \leq l \leq r < N}\, S(l, r)$$

이다. 모든 원소가 음수인 경우에도 빈 부분 배열은 허용되지 않으므로 가장 큰 단일 원소 값이 반환된다.

## 예시

```ts
kadane([-2, 1, -3, 4, -1, 2, 1, -5, 4]); // 6   ([4, -1, 2, 1])
kadane([1]);                              // 1
kadane([5, 4, -1, 7, 8]);                 // 23
kadane([-3, -2, -5, -1, -4]);             // -1  (단일 원소 [-1])
```
