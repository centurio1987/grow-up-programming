# House Robber

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★★★ 상 — 필수 |
| 난이도 | 초급 |

## 함수 인터페이스

```ts
export function houseRobber(nums: number[]): number;
```

## 제약 조건

- $0 \leq N \leq 100{,}000$ (여기서 $N$ 은 `nums` 의 길이, 즉 집의 개수)
- $0 \leq nums[i] \leq 10^4$ (각 집의 현금)

## 문제 상세

일렬로 늘어선 $N$ 개의 집이 있고, $i$ 번째 집에는 $nums[i]$ 만큼의 현금이 있다.

**인접한 두 집은 동시에 선택할 수 없다.** 선택한 집들의 현금 합이 최대가 되도록 할 때, 그 최댓값을 반환하라.

집을 하나도 선택하지 않는 것도 허용되며, 빈 배열인 경우 $0$ 을 반환한다.

## 예시

| `nums` | 반환값 | 설명 |
| --- | --- | --- |
| `[]` | `0` | 선택할 집 없음 |
| `[7]` | `7` | 유일한 집 선택 |
| `[2, 7]` | `7` | 더 큰 집 하나만 선택 |
| `[2, 7, 9, 3, 1]` | `12` | 인덱스 $0, 2, 4$ 선택 ($2 + 9 + 1 = 12$) |
| `[2, 1, 1, 2]` | `4` | 인덱스 $0, 3$ 선택 ($2 + 2 = 4$) |
