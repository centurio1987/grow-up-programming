# Maximum Product Subarray

## 함수 인터페이스

```ts
export function maximumProductSubarray(A: number[]): number;
```

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$ 은 `A` 의 길이)
- $-10 \leq A[i] \leq 10$ (정수, 음수 가능)

## 문제 상세

정수 배열 $A$ 가 주어진다. **연속된 (비어있지 않은) 부분 배열의 곱 중 최댓값** 을 반환한다.

인덱스 구간 $[l, r]$ 의 곱을

$$P(l, r) = \prod_{k=l}^{r} A[k]$$

로 정의하면 결과는

$$\text{maxProduct}(A) = \max_{0 \leq l \leq r < N}\, P(l, r)$$

이다. 음수의 곱이 양수가 될 수 있으므로 부호 변화에 주의해야 한다.

## 예시

| `A` | 반환값 | 설명 |
| --- | --- | --- |
| `[2, 3, -2, 4]` | `6` | $[2, 3]$ |
| `[-2, 0, -1]` | `0` | 음수가 하나뿐, $[0]$ 이 최대 |
| `[-2, 3, -4]` | `24` | 전체 곱 $(-2) \cdot 3 \cdot (-4) = 24$ |
| `[2, -5, -2, -4, 3]` | `24` | $[-2, -4, 3]$ |
| `[0, 2]` | `2` | $[2]$ |
| `[-3]` | `-3` | 단일 원소 |
