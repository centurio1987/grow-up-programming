# Maximum Subarray Sum

## 한 줄 요약

> 함수는 정수 배열을 받아 연속된 부분 배열의 합 중 최댓값을 반환한다.

## 스토리

주식 분석가가 여러 날에 걸친 일별 수익률 데이터를 갖고 있다. 양수는 그날 이익, 음수는 손실을 의미한다. 분석가는 연속된 기간을 선택해 그 기간의 총 수익을 극대화하려 한다.

하루만 선택할 수도 있고, 모든 기간을 통틀어 선택할 수도 있다. 단 선택하는 기간은 반드시 연속된 날들이어야 한다. 모든 날이 손실인 경우에도 그 중 손실이 가장 적은 날 하루를 선택해야 한다.

주어진 데이터에서 연속된 기간을 어떻게 잡아야 총 수익이 가장 커지는지 구하라.

## 함수 인터페이스

```ts
export function maxSubarraySum(nums: number[]): number;
```

- `nums` — 정수 배열 (길이 $N$, 음수 포함 가능)
- 반환 — 연속된 부분 배열의 합 중 최댓값

## 제약 조건

- $1 \leq N \leq 100{,}000$ ($N$은 `nums`의 길이)
- `nums`의 원소는 정수 (음수 포함)
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

정수 배열 `nums`가 주어질 때, 부분 배열 $\text{nums}[i..j]$ ($0 \leq i \leq j < N$)의 합의 최댓값을 반환한다.

$$\text{maxSubarraySum}(\text{nums}) = \max_{0 \leq i \leq j < N} \sum_{k=i}^{j} \text{nums}[k]$$

부분 배열은 최소 하나 이상의 원소를 포함해야 한다. 모든 원소가 음수인 경우에도 가장 큰 단일 원소를 반환한다.

## 예시

```ts
maxSubarraySum([-2, 1, -3, 4, -1, 2, 1, -5, 4]);
// 6 — 부분 배열 [4, -1, 2, 1]의 합

maxSubarraySum([5, 4, -1, 7, 8]);
// 23 — 전체 배열의 합이 최대

maxSubarraySum([-3, -2, -5, -1]);
// -1 — 모두 음수이므로 가장 큰 단일 원소 -1

maxSubarraySum([1]);
// 1 — 원소가 하나뿐

maxSubarraySum([5, -3, 5]);
// 7 — 전체 합 [5, -3, 5]가 단일 원소보다 큼
```
