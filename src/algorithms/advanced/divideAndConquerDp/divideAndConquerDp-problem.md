# 최소 비용 $k$-분할

## 한 줄 요약

> 비용 행렬과 분할 수 $k$를 받아, 구간 $[0, n-1]$을 $k$개의 연속 구간으로 나눴을 때 비용 합의 최솟값을 반환한다.

## 스토리

물류 회사의 배송 구역 설계 담당자 현아는 배송 경로를 여러 구역으로 나누는 일을 맡았다. 배송 노선은 일렬로 늘어선 $n$개의 거점으로 이루어져 있으며, 이 거점들을 $k$개의 연속된 구역으로 묶어야 한다.

각 구역을 담당할 배송 기사를 지정하는 데에는 구역 안에 포함된 거점 조합에 따라 비용이 달라진다. 예를 들어 교통 상황이나 거리 분포에 따라 구간 $[i, j]$를 한 기사에게 맡기는 비용이 사전에 계산된 표로 주어진다.

현아는 이 표를 보고 전체 배송 비용이 가장 적게 드는 구역 분할 방법을 찾아야 한다. 거점 수가 수백 개, 분할 수도 수십 개에 달하더라도 빠르게 답을 구할 수 있어야 한다.

## 함수 인터페이스

```ts
export function divideAndConquerDp(cost: number[][], k: number): number;
```

- `cost` — $n \times n$ 비용 행렬. `cost[i][j]`는 구간 $[i, j]$를 하나의 그룹으로 묶는 비용 ($i \leq j$)
- `k` — 분할할 연속 구간의 수
- 반환 — $k$개의 연속 구간으로 분할했을 때 비용 합의 최솟값

## 제약 조건

- $1 \leq n \leq 500$ (여기서 $n$은 `cost`의 한 변의 길이)
- `cost`는 $n \times n$ 정수 행렬이며 $0 \leq \text{cost}[i][j]$
- $1 \leq k \leq n$
- 비용 행렬은 사각 부등식(quadrangle inequality)을 만족한다고 가정한다
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

$n \times n$ 비용 행렬 `cost[i][j]`가 주어진다. 이 값은 구간 $[i, j]$를 하나의 그룹으로 묶었을 때의 비용이다.

배열의 인덱스 구간 $[0, n-1]$을 정확히 $k$개의 **연속 구간**으로 분할할 때, 각 구간에 해당하는 비용들의 합이 최소가 되도록 하는 분할의 비용 총합을 반환하라.

분할은 겹치지 않고, 빠짐없이, 순서대로 이어져야 한다. $k = 1$이면 전체를 하나의 구간으로 묶어 `cost[0][n-1]`이 답이다. $k = n$이면 각 원소를 단독 구간으로 묶어 $\sum_{i=0}^{n-1} \text{cost}[i][i]$가 답이다.

## 예시

```ts
const c = [
  [0, 2, 5],
  [0, 0, 3],
  [0, 0, 0],
];

divideAndConquerDp(c, 1); // 5 — 전체 구간 [0,2] 하나로 묶음, cost[0][2] = 5
divideAndConquerDp(c, 3); // 0 — 각 원소 단독 구간, cost[0][0]+cost[1][1]+cost[2][2] = 0
divideAndConquerDp(c, 2); // 3 — [0,1]+[2,2] = 2+0=2 또는 [0,0]+[1,2] = 0+3=3, 최솟값 = 2

// 단일 원소
divideAndConquerDp([[7]], 1); // 7 — n=1, k=1, cost[0][0] = 7

// 균등 분할
const c2 = [
  [4, 16, 36, 64],
  [0,  4, 16, 36],
  [0,  0,  4, 16],
  [0,  0,  0,  4],
]; // buildCost([2,2,2,2])
divideAndConquerDp(c2, 2); // 32 — [2,2]+[2,2] = 16+16
```
