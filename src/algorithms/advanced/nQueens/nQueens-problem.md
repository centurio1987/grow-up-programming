# N-Queens

## 한 줄 요약

> $n$을 받아 $n \times n$ 체스판에 $n$개의 퀸을 서로 공격할 수 없게 놓는 방법의 수를 반환한다.

## 스토리

체스 클럽 운영자 도윤은 신입 회원들에게 퀸의 공격 범위를 가르치는 교육 도구를 만들고 있다. 퀸은 같은 행, 같은 열, 같은 대각선 방향으로 얼마든지 이동할 수 있어서 서로를 공격할 수 있다.

도윤은 $n \times n$ 크기의 가상 체스판에 $n$개의 퀸을 올려놓되, 어느 두 퀸도 서로를 공격할 수 없도록 배치하는 예제를 여러 가지 보여주고 싶다. 그러기 위해 먼저 배치 방법이 몇 가지나 있는지를 미리 계산해두어야 한다.

$n$이 커질수록 가능한 배치가 폭발적으로 늘어나기 때문에, 중복 탐색 없이 빠르게 전체 경우의 수를 세는 방법이 필요하다.

## 함수 인터페이스

```ts
export function nQueens(n: number): number;
```

- `n` — 체스판의 한 변의 길이이자 배치할 퀸의 수
- 반환 — $n$개의 퀸을 서로 공격할 수 없게 배치하는 방법의 수

## 제약 조건

- $1 \leq n \leq 12$
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

$n \times n$ 크기의 체스판에 $n$개의 퀸을 배치하되, 어느 두 퀸 $(r_1, c_1)$, $(r_2, c_2)$도 아래 세 조건 중 어느 하나도 만족하지 않아야 한다.

$$r_1 = r_2 \quad \text{(같은 행)}$$
$$c_1 = c_2 \quad \text{(같은 열)}$$
$$|r_1 - r_2| = |c_1 - c_2| \quad \text{(같은 대각선)}$$

이를 만족하는 배치의 수를 모두 세어 반환하라.

- $n = 2, 3$이면 해가 없으므로 0을 반환한다.
- $n = 1$이면 유일하게 하나의 배치만 존재한다.

## 예시

```ts
nQueens(1);  //   1 — 1×1 보드, 유일한 배치
nQueens(2);  //   0 — 해 없음
nQueens(3);  //   0 — 해 없음
nQueens(4);  //   2 — 두 가지 배치 존재
nQueens(5);  //  10
nQueens(6);  //   4
nQueens(7);  //  40
nQueens(8);  //  92 — 고전적인 8-Queens 정답
nQueens(9);  // 352
nQueens(10); // 724
```
