# Radix Sort

## 한 줄 요약

> $0$ 이상의 정수로만 이루어진 배열 `A`를 받아 **오름차순으로 정렬된 새 배열**을 반환한다.

## 스토리

우편물 분류 센터. 오늘 처리해야 할 **`N`건** 소포의 우편번호가 무작위 순서로 넘어왔다.
분류기는 한 번에 숫자 한 자릿수만 읽을 수 있어서, 자릿수를 반복하며 순서를 맞춰나간다.

우편번호는 항상 $0$ 이상의 정수이며, 같은 번호가 여럿 있을 수 있다.
숫자가 딱 하나뿐이어도 정상 처리해야 한다.

## 함수 인터페이스

```ts
export function radixSort(A: number[]): number[];
```

- `A` — 정렬할 비음 정수 배열 (각 원소 $0 \leq A[i] \leq 10^9$)
- 반환 — `A`의 모든 원소를 오름차순으로 정렬한 새 배열

## 제약 조건

- $1 \leq N \leq 100{,}000$ — 여기서 $N$은 배열 `A`의 길이다.
- $0 \leq A[i] \leq 10^9$ — 각 원소는 이 범위의 **비음 정수**다. 음수나 소수는 입력으로 주어지지 않는다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

배열 `A`를 **오름차순(작은 값 → 큰 값)** 으로 정렬한 결과를 새 배열로 반환한다. 다음 규칙을 따른다.

- **값 범위.** 모든 원소가 $[0, 10^9]$ 비음 정수 범위 안에 있다. 최대 10자리 수까지 처리해야 한다.
- **원소 집합 보존.** 반환 배열의 다중집합(multiset)은 입력 `A`의 다중집합과 같아야 한다.
- **빈 배열.** `A`가 비어 있으면 빈 배열을 반환한다.
- **중복 허용.** 같은 값이 여럿이면 그 개수 전부 결과에 포함된다.
- **0 포함.** 0은 유효한 원소이며, $0$이 여럿이면 그 개수만큼 결과 앞자리에 포함된다.

## 예시

```ts
radixSort([170, 45, 75, 90, 802, 24, 2, 66]);
// [2, 24, 45, 66, 75, 90, 170, 802] — 여러 자릿수 혼재

radixSort([5, 4, 3, 2, 1]);
// [1, 2, 3, 4, 5] — 역순 입력

radixSort([0, 10, 0, 1]);
// [0, 0, 1, 10] — 0 포함(경계: 0이 여럿)

radixSort([12345]);
// [12345] — 원소가 하나(경계: N=1)

radixSort([7, 7, 7]);
// [7, 7, 7] — 전부 동일(경계: 모든 값 같음)

radixSort([1_000_000_000, 0, 999_999_999, 1]);
// [0, 1, 999999999, 1000000000] — 최댓값 포함(경계: A[i]=10^9)

radixSort([]);
// [] — 빈 배열(경계: N=0)
```
