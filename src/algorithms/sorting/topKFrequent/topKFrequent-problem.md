# Top K Frequent Elements

## 한 줄 요약

> 정수 배열 `A`와 정수 `k`가 주어질 때, **`A`에서 등장 빈도가 가장 높은 상위 `k`개의 원소**를 빈도 내림차순으로 반환한다.

## 스토리

음악 스트리밍 서비스 차트팀. 오늘 하루 **`N`건**의 재생 기록이 곡 ID 배열로 들어왔다.
담당자가 말한다. "오늘 가장 많이 재생된 곡 `k`개만 뽑아주세요."

같은 곡이 여러 번 등장하면 그만큼 인기 있는 것이다.
빈도 동률이 생기면 어느 곡을 먼저 뽑아도 무방하다.

## 함수 인터페이스

```ts
export function topKFrequent(A: number[], k: number): number[];
```

- `A` — 정수 배열 (각 원소 $-10^9 \leq A[i] \leq 10^9$)
- `k` — 상위 몇 개를 반환할지 지정하는 양의 정수
- 반환 — 빈도 내림차순으로 정렬된 상위 `k`개 원소 배열

## 제약 조건

- $1 \leq N \leq 100{,}000$ — 여기서 $N$은 배열 `A`의 길이다.
- $-10^9 \leq A[i] \leq 10^9$ — 각 원소는 이 범위의 정수이며 음수일 수 있다.
- $1 \leq k \leq |\text{고유값 집합}|$ — `k`는 항상 `A`에 등장하는 서로 다른 값의 개수 이하다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

원소 $v$의 빈도를 다음과 같이 정의한다.

$$f(v) = \bigl|\{ i \mid A[i] = v \}\bigr|$$

반환 배열 $R$은 다음 조건을 만족해야 한다.

- $|R| = k$
- $\forall u \notin R,\ \forall v \in R: f(v) \geq f(u)$ — $R$에 속하지 않는 모든 원소보다 $R$의 원소들의 빈도가 크거나 같다.
- **출력 순서.** 빈도 내림차순으로 정렬한다. 빈도가 같은 원소끼리의 상대 순서는 임의로 결정해도 된다.
- **음수 포함.** 음수 원소도 일반 정수로 취급하며 빈도 계산에 포함된다.
- **단일 원소.** `A`의 길이가 1이고 `k`가 1이면 그 원소 하나를 담은 배열을 반환한다.

## 예시

```ts
topKFrequent([1, 1, 1, 2, 2, 3], 2);
// [1, 2] — 빈도: 1→3회, 2→2회, 3→1회 → 상위 2개

topKFrequent([1], 1);
// [1] — 원소가 하나(경계: N=1, k=1)

topKFrequent([-1, -1, -1, 2, 2, 0], 2);
// [-1, 2] — 음수 포함: -1→3회, 2→2회, 0→1회 → 상위 2개

topKFrequent([7, 7, 7, 7], 1);
// [7] — 전부 동일(경계: 모든 값 같음)

topKFrequent([4, 4, 4, 4, 2, 2, 2, 1, 1, 3], 3);
// [4, 2, 1] — 빈도: 4→4회, 2→3회, 1→2회, 3→1회 → 상위 3개

topKFrequent([1_000_000_000, 1_000_000_000, -1_000_000_000], 1);
// [1000000000] — 값 범위 경계: 10^9→2회, -10^9→1회 → 상위 1개
```
