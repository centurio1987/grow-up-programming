# 다각형 면적

## 한 줄 요약

> 꼭짓점 목록으로 정의된 단순 다각형의 면적을 반환한다.

## 스토리

지적 측량사 미준은 주민들로부터 토지 경계선 좌표 목록을 받아 면적을 계산해 달라는 의뢰를 받는다. 토지 경계는 직선으로만 이어진 다각형 형태이며, 꼭짓점 순서는 시계 방향일 수도, 반시계 방향일 수도 있다.

미준에게는 꼭짓점의 나열 순서가 달라도, 오목한 모양이어도 동일한 면적을 정확히 내놓는 계산 함수가 필요하다.

주어진 꼭짓점 배열로 다각형의 면적을 계산하는 함수를 만들어라.

## 함수 인터페이스

```ts
export type Point = [number, number];

export function polygonArea(polygon: Point[]): number;
```

- `polygon` — 다각형의 꼭짓점 배열. 각 원소는 `[x, y]` 형태의 2D 좌표. 마지막 꼭짓점과 첫 번째 꼭짓점이 자동으로 연결되어 닫힌 다각형을 이룬다.
- 반환 — 다각형의 면적. 항상 $\geq 0$ 인 부동소수로 반환한다.

## 제약 조건

- $3 \leq N \leq 10^5$ — 여기서 $N$ 은 꼭짓점 배열의 길이
- $-10^9 \leq x, y \leq 10^9$ — 각 꼭짓점 좌표의 범위
- `polygon` 은 자기교차가 없는 단순 다각형 (볼록·오목 모두 가능)
- 꼭짓점 순서는 시계 방향과 반시계 방향 모두 유효한 입력이다
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

정점 $v_0, v_1, \ldots, v_{n-1}$ 으로 정의된 단순 다각형의 면적은 다음 신발끈 공식(Shoelace Formula)으로 계산한다.

$$A = \frac{1}{2} \left| \sum_{i=0}^{n-1} (x_i\, y_{i+1} - x_{i+1}\, y_i) \right|$$

여기서 인덱스는 순환적이며 $v_n = v_0$ 이다.

- **절댓값을 취한다.** 따라서 꼭짓점 순서가 시계 방향이든 반시계 방향이든 동일한 비음수 결과를 반환한다.
- 모든 꼭짓점이 일직선상에 있으면 넓이는 0이다.
- 공선점(변 위에 있는 중간 꼭짓점)이 포함되어도 올바른 면적을 반환한다.

## 예시

```ts
// 2×2 정사각형
polygonArea([[0, 0], [2, 0], [2, 2], [0, 2]]); // 4 — 가로 2 × 세로 2

// 밑변 4, 높이 3인 직각삼각형
polygonArea([[0, 0], [4, 0], [0, 3]]); // 6 — (4 × 3) / 2

// 시계 방향 입력도 동일한 면적
polygonArea([[0, 0], [0, 2], [2, 2], [2, 0]]); // 4 — 방향과 무관하게 절댓값 반환

// 오목 다각형 (L자, 4×4에서 2×2 잘라낸 모양)
polygonArea([[0, 0], [4, 0], [4, 2], [2, 2], [2, 4], [0, 4]]); // 12

// 공선점 포함 — 변 위의 중간 꼭짓점
polygonArea([[0, 0], [2, 0], [4, 0], [4, 4], [0, 4]]); // 16 — (2,0)은 변 위의 중간 점이지만 면적에 영향 없음

// 모든 꼭짓점이 일직선
polygonArea([[0, 0], [1, 0], [2, 0]]); // 0 — 넓이 없음

// 최소 삼각형
polygonArea([[0, 0], [1, 0], [0, 1]]); // 0.5
```
