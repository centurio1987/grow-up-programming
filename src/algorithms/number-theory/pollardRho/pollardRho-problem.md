# 비자명한 약수 찾기 (Pollard's Rho)

## 한 줄 요약

> 함수는 합성수 `n`을 받아 1도 `n`도 아닌 약수 하나를 반환한다.

## 스토리

보안 연구원이 오래된 RSA 모듈러스를 분석하고 있다. 모듈러스 $n$이 두 소수 $p \times q$의 곱이라면 암호를 해독할 수 있다. $n$이 20자리 수라면 시행 나눗셈으로 약수를 찾으려면 $10^{10}$번 이상 나눠야 해서 현실적이지 않다.

연구원은 $n$의 비자명한 약수, 즉 1이나 $n$이 아닌 약수를 하나만 찾으면 된다. 나머지 분해는 이 약수를 재귀적으로 적용하면 된다.

$n$이 짝수면 즉시 2를 반환할 수 있다. $n$이 소수이면 비자명한 약수가 존재하지 않으므로 $n$ 자체를 반환한다.

## 함수 인터페이스

```ts
export function pollardRho(n: bigint): bigint;
```

- `n` — 약수를 찾을 정수, $n \geq 2$인 bigint
- 반환 — $n$의 약수 $d$, $1 < d \leq n$; 합성수이면 $1 < d < n$, 소수이면 $d = n$

## 제약 조건

- $n \geq 2$ (bigint)
- 합성수 입력 권장; 소수 입력 시 $n$ 자체 반환 허용
- $n$이 짝수이면 즉시 $2$를 반환해도 됨
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

$n$의 비자명한 약수 $d$ ($1 < d < n$)를 하나 반환한다.

- 반환값 $d$는 반드시 $n \bmod d = 0$을 만족해야 한다.
- 여러 비자명한 약수가 있을 때 어느 것을 반환해도 무방하다.
- $n$이 소수이면 $d = n$을 반환한다.
- $n$이 짝수이면 $d = 2$를 반환한다.

## 예시

```ts
pollardRho(15n);      // 3n 또는 5n  — 15 = 3 × 5, 둘 중 하나
pollardRho(8n);       // 2n          — 짝수이므로 2
pollardRho(49n);      // 7n          — 49 = 7 × 7
pollardRho(91n);      // 7n 또는 13n — 91 = 7 × 13
pollardRho(10403n);   // 101n 또는 103n — 10403 = 101 × 103
pollardRho(1_000_003n); // 1_000_003n — 소수이면 자기 자신
```
