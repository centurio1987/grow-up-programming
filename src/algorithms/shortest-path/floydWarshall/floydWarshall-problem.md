# Floyd-Warshall 모든 쌍 최단 경로

## 한 줄 요약

> 함수는 음수 간선을 포함할 수 있는 그래프를 받아 모든 정점 쌍 사이의 최단 거리를 담은 2차원 배열을 반환한다.

## 스토리

도시 계획 연구원 태연은 도시 내 모든 지역 간 이동 비용을 분석해야 한다. 어떤 구간은 보조금이 지급되어 음수 비용이 발생하기도 한다. 단, 전체 구간을 순환해도 계속 이득이 발생하는 구간은 없다고 가정한다.

태연이 필요한 것은 특정 두 지점 간의 경로가 아니라, 모든 지역 쌍 사이의 최솟값 표다. 이 표가 있으면 어떤 출발지와 목적지의 조합이든 즉시 최소 비용을 조회할 수 있다.

지역 수가 수백 개이므로, 모든 쌍의 최단 거리를 한 번에 계산하는 방법이 필요하다.

## 함수 인터페이스

```ts
export function floydWarshall(
  n: number,
  edges: [number, number, number][],
): number[][];
```

- `n` — 정점 수. 정점 번호는 $0$부터 $n - 1$.
- `edges` — 방향 간선 목록. 각 원소는 $[u, v, w]$ 형태이며 가중치 $w$ (음수 허용)인 방향 간선 $u \to v$.
- 반환 — 크기 $n \times n$인 2차원 배열 $D$. $D[u][v]$는 $u \to v$의 최단 거리. 도달할 수 없는 쌍은 `Infinity`. $D[v][v] = 0$.

## 제약 조건

- $1 \leq V \leq 500$ (정점 수)
- $0 \leq E \leq V \cdot (V - 1)$ (간선 수)
- $-10^6 \leq w(u, v) \leq 10^6$ (가중치; 음수 허용)
- 음수 사이클은 존재하지 않는다
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

음수 가중치를 포함할 수 있는 방향 그래프 $G = (V, E)$가 주어진다 (단, 음수 사이클 없음). 모든 정점 쌍 $(u, v)$에 대한 최단 거리 $d(u, v)$를 구하라.

최단 거리는 경로 위 간선 가중치의 합이 최소인 경로 비용으로 정의된다.

- 모든 $v$에 대해 $D[v][v] = 0$이다.
- 도달할 수 없는 쌍의 거리는 `Infinity`다.
- 동일한 두 정점 사이에 다중 간선이 있으면 더 작은 가중치를 사용한다.
- 방향 그래프이므로 $D[u][v] \neq D[v][u]$일 수 있다.

## 예시

```ts
// 음수 간선 포함, 간접 경로가 더 짧은 경우
const edges: [number, number, number][] = [
  [0, 1, 3],
  [1, 2, 2],
  [0, 2, 10],
  [2, 0, 4],
];
floydWarshall(3, edges);
// [
//   [0, 3, 5],   — 0→2: 0→1→2 = 5 (직접 간선 10보다 짧음)
//   [6, 0, 2],   — 1→0: 1→2→0 = 6
//   [4, 7, 0],   — 2→1: 2→0→1 = 7
// ]

// 도달 불가능한 쌍은 Infinity
floydWarshall(3, [[0, 1, 1]]);
// [
//   [0,        1,        Infinity],
//   [Infinity, 0,        Infinity],
//   [Infinity, Infinity, 0       ],
// ]

// 간선이 없는 그래프: 대각선만 0
floydWarshall(2, []);
// [
//   [0,        Infinity],
//   [Infinity, 0       ],
// ]

// 방향성: u→v와 v→u 거리는 독립적
floydWarshall(2, [[0, 1, 3], [1, 0, 7]]);
// [
//   [0, 3],
//   [7, 0],
// ]
```
