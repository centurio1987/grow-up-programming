# 최대 유량 (Maximum Flow)

## 한 줄 요약

> 유량 네트워크의 정점 수 `n`, 방향 간선 목록 `edges`, 소스 `source`, 싱크 `sink` 를 받아, 소스에서 싱크로 흘릴 수 있는 **최대 유량**을 `{ flow }` 형태로 반환한다.

## 스토리

도시 외곽 물류 센터에서 항구까지 여러 경로로 화물을 보낼 수 있다. 각 도로마다 한 번에 통과할 수 있는 화물 트럭 대수(용량)가 다르다. 어떤 경유 도시를 거쳐 가더라도 들어온 트럭 수와 나가는 트럭 수가 같아야 한다(유량 보존). 물류 센터에서 출발한 트럭이 항구에 도달하는 수가 곧 이익이다.

물류 팀장은 묻는다. "이 도로망에서 한 시간에 최대 몇 대를 항구까지 보낼 수 있죠?"

트럭을 보낼 때 일부 경로에서 이미 지나가던 트럭을 다른 경로로 돌릴 수 있다면 전체 수송량을 더 늘릴 수도 있다. 네트워크 전체를 보고 가장 많은 트럭을 항구까지 보내는 방법을 찾아야 한다.

## 함수 인터페이스

```ts
export function maxFlow(
  n: number,
  edges: [number, number, number][],
  source: number,
  sink: number,
): { flow: number };
```

- `n` — 정점의 개수. 정점 번호는 $0 \ldots n-1$.
- `edges` — 방향 가중치 간선 목록. 각 원소 `[u, v, c]` 는 정점 `u` 에서 `v` 로 향하는 용량 `c` 의 간선이다.
- `source` — 소스 정점 번호.
- `sink` — 싱크 정점 번호.
- 반환 — `{ flow }`. 소스에서 싱크로의 최대 유량.

## 제약 조건

- $2 \leq n \leq 500$
- $0 \leq E \leq 10^4$ (여기서 $E$ 는 간선의 수)
- $0 \leq c(u, v) \leq 10^6$
- $0 \leq \text{source},\, \text{sink} < n,\;\; \text{source} \neq \text{sink}$
- 정점 번호는 $0 \ldots n-1$.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

유량 네트워크 $G = (V, E, c)$ 와 소스 $s$, 싱크 $t$ 가 주어졌을 때, 다음 조건을 모두 만족하는 유량 함수 $f: E \to \mathbb{R}_{\geq 0}$ 의 총 유량을 최대화한다.

- 용량 제약: $0 \leq f(u, v) \leq c(u, v)$
- 유량 보존: $\forall v \in V \setminus \{s, t\},\; \sum_{u} f(u, v) = \sum_{w} f(v, w)$

총 유량:

$$|f| = \sum_{v:\, (s, v) \in E} f(s, v) - \sum_{u:\, (u, s) \in E} f(u, s)$$

$$\text{maxFlow}(G, s, t) = \max_{f \text{ feasible}} |f|$$

- 소스에서 싱크로 향하는 경로가 없으면 유량은 `0` 이다.
- 평행 간선(같은 두 정점 사이의 여러 간선)은 용량이 합산되는 효과를 낸다.
- 용량 `0` 인 간선은 유량을 흘릴 수 없다.
- 자기 루프는 유량에 영향을 주지 않는다.

## 예시

```ts
maxFlow(
  4,
  [[0, 1, 3], [0, 2, 2], [1, 2, 1], [1, 3, 2], [2, 3, 3]],
  0, 3,
); // { flow: 5 } — 경로 0→1→3으로 2, 경로 0→2→3으로 2, 경로 0→1→2→3으로 1

maxFlow(2, [[0, 1, 10]], 0, 1); // { flow: 10 } — 소스와 싱크가 직접 연결, 용량이 곧 최대 유량

maxFlow(3, [[0, 1, 5]], 0, 2); // { flow: 0 } — 정점 1과 싱크 2 사이 간선 없어 경로 없음

maxFlow(2, [], 0, 1); // { flow: 0 } — 간선 없음

maxFlow(
  4,
  [[0, 1, 3], [0, 2, 3], [1, 2, 2], [1, 3, 3], [2, 3, 3]],
  0, 3,
); // { flow: 6 } — 역방향 간선 활용 시 최대 유량 6

maxFlow(3, [[0, 1, 1_000_000], [1, 2, 1_000_000]], 0, 2); // { flow: 1000000 } — 직렬 경로, 병목 용량이 최대 유량
```
