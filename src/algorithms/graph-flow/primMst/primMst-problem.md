# Prim 최소 신장 트리

## 한 줄 요약

> 가중치 무방향 그래프의 정점 수 `n` 과 간선 목록 `edges` 를 받아, 모든 정점을 연결하는 간선 부분집합의 **가중치 합이 최소인 신장 트리**의 합계를 반환한다. 그래프가 연결되지 않으면 `-1` 을 반환한다.

## 스토리

신도시 개발 계획. 여러 구역을 하나의 도로망으로 연결해야 한다. 구역 사이에는 도로 건설 후보 경로가 여러 개 있고, 각 경로마다 공사비가 다르다. 목표는 **모든 구역이 도로로 연결되면서 공사비 합계가 가장 적게** 드는 도로망을 찾는 것이다.

이미 이어진 구역에서 이웃한 미연결 구역으로 한 칸씩 뻗어 나가는 방식으로 도로를 만들어 간다. 연결할 때마다 아직 연결되지 않은 구역과 이을 수 있는 경로 중 공사비가 가장 싼 경로를 고른다. 이 과정을 모든 구역이 연결될 때까지 반복하면 최소 공사비 도로망이 완성된다.

구역이 여러 덩어리로 분리되어 있어 전체를 하나로 이을 수 없다면, 도로망 완성이 불가능하다는 신호를 돌려준다.

## 함수 인터페이스

```ts
export function primMst(n: number, edges: [number, number, number][]): number;
```

- `n` — 정점의 개수. 정점 번호는 $0 \ldots n-1$.
- `edges` — 무방향 가중치 간선 목록. 각 원소 `[u, v, w]` 는 정점 `u` 와 `v` 를 잇는 가중치 `w` 의 간선이다.
- 반환 — 최소 신장 트리의 가중치 합. 그래프가 연결되지 않아 신장 트리를 구성할 수 없으면 `-1`.

## 제약 조건

- $1 \leq n \leq 10^4$
- $0 \leq E \leq n \cdot (n-1) / 2$ (여기서 $E$ 는 간선의 수)
- $0 \leq w(u, v) \leq 10^9$
- 정점 번호는 $0 \ldots n-1$.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

가중치 무방향 그래프 $G = (V, E, w)$ 에서 신장 트리 $T \subseteq E$ 는 모든 정점을 포함하고 사이클이 없는 간선 부분집합이다. 최소 신장 트리는 가중치 합이 가장 작은 신장 트리다.

$$W(T) = \sum_{(u, v) \in T} w(u, v), \qquad \text{primMst}(G) = \min_{T \in \mathcal{T}(G)} W(T)$$

여기서 $\mathcal{T}(G)$ 는 $G$ 의 모든 신장 트리의 집합이다.

- 신장 트리는 정확히 $n - 1$ 개의 간선으로 이루어진다.
- 그래프가 연결되지 않으면 신장 트리를 구성할 수 없으므로 `-1` 을 반환한다.
- 정점이 1개인 경우 신장 트리는 간선 없이 정점 하나만으로 구성되며, 가중치 합은 `0` 이다.
- 자기 루프 `[v, v, w]` 는 신장 트리에 포함될 수 없으므로 무시한다.
- 평행 간선(같은 두 정점을 잇는 여러 간선)이 있을 경우 그 중 가중치가 작은 것만 고려된다.

## 예시

```ts
primMst(4, [
  [0, 1, 1],
  [1, 2, 2],
  [2, 3, 3],
  [0, 3, 10],
]); // 6 — 간선 (0,1)=1, (1,2)=2, (2,3)=3 선택; (0,3)=10은 포함하면 사이클

primMst(3, [[0, 1, 5]]); // -1 — 정점 2가 어떤 간선과도 연결되지 않아 신장 트리 불가

primMst(1, []); // 0 — 정점이 1개이면 간선 없이 신장 트리 완성, 합 = 0

primMst(2, [[0, 1, 100], [0, 1, 1]]); // 1 — 평행 간선 중 가중치가 작은 것을 선택

primMst(2, [[0, 0, 100], [0, 1, 5]]); // 5 — 자기 루프는 신장 트리에 포함되지 않음

primMst(3, [[0, 1, 0], [1, 2, 5]]); // 5 — 가중치 0 간선도 정상 처리
```
