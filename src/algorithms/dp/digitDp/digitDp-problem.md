# 자릿수 합이 K인 수의 개수

## 한 줄 요약

> 함수는 상한 N과 목표 합 K를 받아, 1 이상 N 이하의 정수 중 십진 자릿수의 합이 정확히 K인 수의 개수를 반환한다.

## 스토리

품질 관리 팀의 태호는 생산 일련번호 중 특별한 성질을 갖는 것만 골라내는 검사 도구를 만들고 있다. 일련번호는 1부터 시작하며, 공장 규정에 따라 각 자릿수를 모두 더한 값이 특정 숫자와 일치하는 제품에는 별도 마킹이 필요하다.

현재 배치의 마지막 일련번호 N이 정해지면, 1부터 N까지 번호 중 자릿수 합이 정확히 K인 것이 몇 개인지 빠르게 알아야 한다. 태호의 공장에서는 N이 10의 15제곱까지 커질 수 있어서, 일일이 세는 방식으로는 시간이 부족하다.

## 함수 인터페이스

```ts
export function digitDp(N: number, K: number): number;
```

- `N` — 범위의 상한 ($1$ 이상 $10^{15}$ 이하의 양의 정수)
- `K` — 목표 자릿수 합 ($0$ 이상 $135$ 이하)
- 반환 — $1$ 이상 $N$ 이하의 정수 중 십진 자릿수 합이 정확히 $K$인 수의 개수

## 제약 조건

- $1 \leq N \leq 10^{15}$
- $0 \leq K \leq 135$ ($15$자리 수의 자릿수 합 최대값은 $15 \times 9 = 135$)
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

자릿수 합이란 수를 십진법으로 표기했을 때 각 자리의 숫자를 모두 더한 값이다. 예컨대 $123$의 자릿수 합은 $1 + 2 + 3 = 6$이다.

$K = 0$인 경우, $1$ 이상의 정수 중 자릿수 합이 $0$인 수는 없으므로 $0$을 반환한다.

$K$가 $N$의 자릿수 개수 $\times 9$보다 크면 가능한 수가 없으므로 $0$을 반환한다.

## 예시

```ts
digitDp(9, 5);
// 1 — 1~9 중 자릿수 합이 5인 수: 5

digitDp(100, 1);
// 3 — 1~100 중 자릿수 합이 1인 수: 1, 10, 100

digitDp(20, 2);
// 3 — 1~20 중 자릿수 합이 2인 수: 2, 11, 20

digitDp(99, 18);
// 1 — 두 자리 수 중 자릿수 합이 18인 수: 99

digitDp(100, 0);
// 0 — 1 이상에서 자릿수 합이 0인 수는 없음
```
