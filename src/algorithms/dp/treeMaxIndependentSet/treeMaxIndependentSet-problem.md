# 트리 최대 가중 독립집합

## 한 줄 요약

> 함수는 가중 트리의 노드 수, 간선 목록, 노드별 가중치를 받아, 인접한 두 노드를 동시에 선택하지 않는 부분집합 중 가중치 합의 최댓값을 반환한다.

## 스토리

도시 계획가 혜원은 나무 형태로 연결된 지역 네트워크에 기지국을 설치하려 한다. 기지국끼리 직접 연결된 위치에 함께 설치하면 전파 간섭이 생기므로, 인접한 두 지점에는 동시에 설치할 수 없다. 각 지점은 설치 시 얻는 커버리지 점수(가중치)가 다르며, 음수 점수인 지점은 설치하지 않는 편이 낫다.

혜원은 간섭 없이 얻을 수 있는 커버리지 점수의 합을 최대로 만들고 싶다. 아무 지점도 선택하지 않는 것도 항상 선택지로 유효하다.

## 함수 인터페이스

```ts
export function treeMaxIndependentSet(
  n: number,
  edges: [number, number][],
  weights: number[],
): number;
```

- `n` — 노드 수 ($1 \leq n \leq 10^5$)
- `edges` — 트리의 간선 목록 (길이 $n-1$의 무방향 간선 배열, 노드 번호 $0$~$n-1$, 루트는 $0$)
- `weights` — 각 노드의 가중치 (`weights[i]`는 노드 $i$의 가중치)
- 반환 — 어떤 두 인접 노드도 동시에 포함하지 않는 부분집합 중 가중치 합의 최댓값

## 제약 조건

- $1 \leq n \leq 10^5$
- $-10^4 \leq weights[i] \leq 10^4$
- `edges`의 길이는 $n-1$이며, 주어진 그래프는 연결된 무방향 트리이다.
- 노드 번호는 $0$부터 $n-1$까지이며 루트는 $0$이다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

독립집합이란 어떤 두 원소도 트리 상에서 인접(간선으로 직접 연결)하지 않는 노드들의 집합이다.

빈 집합도 독립집합으로 인정하며 합은 $0$이다. 따라서 가중치가 모두 음수라면 $0$을 반환한다.

가중치가 음수인 노드는 선택하지 않는 것이 유리할 수 있다.

## 예시

```ts
treeMaxIndependentSet(1, [], [5]);
// 5 — 단일 노드이므로 그대로 선택

treeMaxIndependentSet(1, [], [-3]);
// 0 — 음수 가중치이므로 선택하지 않는 빈 집합이 최적

treeMaxIndependentSet(3, [[0, 1], [0, 2]], [1, 2, 3]);
// 5 — 노드 1(가중치 2)과 노드 2(가중치 3)을 선택; 노드 0과 인접하지 않음

treeMaxIndependentSet(4, [[0, 1], [1, 2], [2, 3]], [10, 1, 1, 10]);
// 20 — 노드 0(10)과 노드 3(10)을 선택; 서로 인접하지 않음

treeMaxIndependentSet(3, [[0, 1], [1, 2]], [-1, -2, -3]);
// 0 — 모든 가중치가 음수이므로 아무것도 선택하지 않음
```
