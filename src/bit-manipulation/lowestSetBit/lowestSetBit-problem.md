# 최하위 1비트 (Lowest Set Bit)

## 한 줄 요약

> 32비트 정수를 받아, 가장 낮은 위치의 1비트만 남기고 나머지를 0으로 만든 값을 반환한다.

## 스토리

항법 엔지니어 민준은 선박의 GPS 좌표 오류를 수정하는 소프트웨어를 개발한다. 오류 코드는 비트마스크로 표현되며, 여러 오류가 동시에 발생했을 때 가장 우선순위가 높은 오류(가장 낮은 비트)를 먼저 처리해야 한다.

오류 코드 `x`에서 가장 낮은 비트 위치의 오류만 꺼내, 그 오류 하나만 담은 새 코드를 만들어야 한다. 나머지 오류는 차례로 처리하기 위해 그대로 남겨 둔다.

## 함수 인터페이스

```ts
export function lowestSetBit(x: number): number;
```

- `x` — 임의의 32비트 정수
- 반환 — `x`에서 가장 낮은 위치의 1비트만 남긴 값. `x === 0`이면 `0`을 반환.

## 제약 조건

- $-2^{31} \leq x \leq 2^{31} - 1$ (32비트 정수)
- $x = 0$이면 `0`을 반환한다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

32비트 정수 $x$가 주어질 때, $x$의 이진 표현에서 가장 낮은 자리에 있는 1비트만 남기고 그 외 모든 비트를 0으로 만든 값을 반환한다.

예를 들어 $x = 12 = 0\text{b}00001100$이면 가장 낮은 1비트는 2번 위치(값 4)이므로 결과는 $4 = 0\text{b}00000100$이다.

$x = 0$이면 1비트가 없으므로 $0$을 반환한다.

## 예시

```ts
lowestSetBit(12);     //  4 — 0b1100에서 가장 낮은 1비트는 0b0100 = 4
lowestSetBit(6);      //  2 — 0b0110에서 가장 낮은 1비트는 0b0010 = 2
lowestSetBit(1);      //  1 — 0b0001, 최하위 비트가 유일한 1비트
lowestSetBit(16);     // 16 — 0b10000, 단일 비트이므로 자기 자신
lowestSetBit(0);      //  0 — 1비트가 없음
lowestSetBit(-1);     //  1 — 2의 보수 표현에서 -1은 모든 비트가 1, 최하위는 값 1
lowestSetBit(-12);    //  4 — 2의 보수에서 -12의 최하위 1비트는 4
```
