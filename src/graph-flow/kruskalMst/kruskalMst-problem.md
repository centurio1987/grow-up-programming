# Kruskal 최소 신장 트리

## 한 줄 요약

> 가중치 무방향 그래프의 정점 수 `n` 과 간선 목록 `edges` 를 받아, 모든 정점을 연결하는 간선 부분집합의 **가중치 합이 최소인 신장 트리**의 합계를 반환한다. 그래프가 연결되지 않으면 `-1` 을 반환한다.

## 스토리

산간 마을 다섯 곳에 전력망을 깔아야 한다. 마을 사이를 잇는 전선 후보 경로가 여러 개 있고, 각 경로마다 공사비가 다르다. 예산이 빠듯하니 모든 마을에 전기가 들어오는 것을 보장하면서 **공사비 총합을 최소로** 줄여야 한다.

중요한 제약이 있다. 전력망은 트리 형태여야 한다. 어딘가에서 전선이 두 번 연결되는 구간이 생기면 전기는 오가지만 그 경로는 낭비다. 따라서 마을 수보다 딱 하나 적은 전선 구간만 선택해야 전기도 흐르고 비용도 아낀다.

어떤 마을은 다른 마을과 연결 가능한 후보 경로가 아예 없을 수도 있다. 이런 경우는 전력망을 완성할 수 없으므로 불가능 신호를 돌려준다.

## 함수 인터페이스

```ts
export function kruskalMst(n: number, edges: [number, number, number][]): number;
```

- `n` — 정점의 개수. 정점 번호는 $0 \ldots n-1$.
- `edges` — 무방향 가중치 간선 목록. 각 원소 `[u, v, w]` 는 정점 `u` 와 `v` 를 잇는 가중치 `w` 의 간선을 나타낸다.
- 반환 — 최소 신장 트리의 가중치 합. 그래프가 연결되지 않아 신장 트리를 구성할 수 없으면 `-1`.

## 제약 조건

- $1 \leq n \leq 10^5$
- $0 \leq E \leq 2 \cdot 10^5$ (여기서 $E$ 는 간선의 수)
- $0 \leq w(u, v) \leq 10^9$
- 정점 번호는 $0 \ldots n-1$.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

가중치 무방향 그래프 $G = (V, E, w)$ 에서 신장 트리 $T \subseteq E$ 는 모든 정점을 포함하고 사이클이 없는 간선 부분집합이다. 최소 신장 트리는 가중치 합이 가장 작은 신장 트리다.

$$W(T) = \sum_{(u, v) \in T} w(u, v)$$

$$\text{kruskalMst}(G) = \min_{T \in \mathcal{T}(G)} W(T)$$

여기서 $\mathcal{T}(G)$ 는 $G$ 의 모든 신장 트리의 집합이다.

- 신장 트리는 정확히 $n - 1$ 개의 간선으로 이루어진다.
- 그래프가 연결되어 있지 않아 $n - 1$ 개의 간선을 선택하는 것이 불가능하면 `-1` 을 반환한다.
- 정점이 1개인 경우 신장 트리는 간선 없이 정점 하나만으로 구성되며, 가중치 합은 `0` 이다.
- 동일 가중치 간선이 여럿이거나 평행 간선(같은 두 정점을 잇는 여러 간선)이 있어도 올바르게 처리한다.

## 예시

```ts
kruskalMst(4, [
  [0, 1, 1],
  [1, 2, 2],
  [2, 3, 3],
  [0, 3, 10],
]); // 6 — 간선 (0,1)=1, (1,2)=2, (2,3)=3 선택; (0,3)=10은 사이클이 생겨 제외

kruskalMst(3, [
  [0, 1, 5],
]); // -1 — 정점 2가 어떤 간선과도 연결되지 않아 신장 트리 불가

kruskalMst(1, []); // 0 — 정점이 1개이면 간선 없이 신장 트리 완성, 합 = 0

kruskalMst(2, [[0, 1, 100], [0, 1, 1]]); // 1 — 평행 간선 중 가중치가 작은 것을 선택

kruskalMst(4, [[0, 1, 1], [1, 2, 1], [2, 3, 1], [0, 3, 1]]); // 3 — 동일 가중치 4개 중 사이클 안 만드는 3개 선택
```
