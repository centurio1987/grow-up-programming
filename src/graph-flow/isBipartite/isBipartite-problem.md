# 이분 그래프 판정 (Bipartite Check)

## 한 줄 요약

> 무방향 그래프를 나타내는 정점 수 `n` 과 간선 목록 `edges` 를 받아, 그래프가 **이분 그래프**이면 `true`, 아니면 `false` 를 반환한다.

## 스토리

학교 동아리 임원 선발 위원회. 투표 용지를 한 장씩 들고 회의실에 모인 사람들이 있다. 위원회 규정상, 같은 쪽 사람끼리는 서로 아는 관계가 있으면 안 된다 — 서로 아는 두 사람은 반드시 **다른 쪽**에 속해야 공정한 평가가 이뤄진다.

총무는 참석자들이 "서로 안다"는 관계를 쭉 적어왔다. 이제 이 사람들을 두 그룹으로 나눌 수 있는지 확인해야 한다. 두 그룹으로 나눴을 때 **그룹 내부에 서로 아는 쌍이 없어야** 규정을 통과한다.

아는 관계가 전혀 없는 사람은 어느 쪽에 넣어도 상관없다. 여러 개의 독립된 소그룹이 있어도 각 소그룹마다 조건을 만족해야 전체가 통과된다.

## 함수 인터페이스

```ts
export function isBipartite(n: number, edges: [number, number][]): boolean;
```

- `n` — 정점의 개수. 정점 번호는 $0 \ldots n-1$.
- `edges` — 무방향 간선 목록. 각 원소 `[u, v]` 는 정점 `u` 와 `v` 사이에 간선이 있음을 의미한다.
- 반환 — 그래프가 이분 그래프이면 `true`, 아니면 `false`.

## 제약 조건

- $1 \leq n \leq 10^5$
- $0 \leq E \leq 2 \cdot 10^5$ (여기서 $E$ 는 간선의 수)
- 그래프가 연결되어 있지 않을 수 있다. 모든 컴포넌트에 대해 판정한다.
- 정점 번호는 $0 \ldots n-1$.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

무방향 그래프 $G = (V, E)$ 가 이분 그래프인지 판정한다.

이분 그래프란, 정점 집합을 서로소인 두 부분 $A$, $B$ 로 분할하여 모든 간선이 $A$ 와 $B$ 사이를 잇도록 만들 수 있는 그래프다. 즉:

$$\exists\, (A, B):\; V = A \cup B,\; A \cap B = \emptyset,\; \forall (u, v) \in E,\; (u \in A \land v \in B) \lor (u \in B \land v \in A)$$

- 그래프가 연결되어 있지 않을 경우, **모든 연결 컴포넌트**가 이분 그래프여야 `true` 를 반환한다.
- 간선이 없는 고립 정점은 어느 쪽 집합에도 넣을 수 있으므로 이분 조건을 위반하지 않는다.
- 자기 루프 `[v, v]` 가 있으면 이분 그래프가 될 수 없다.

## 예시

```ts
isBipartite(4, [[0, 1], [1, 2], [2, 3], [3, 0]]); // true  — 길이 4짜리 사이클, 두 집합 {0,2}와 {1,3}으로 분할 가능
isBipartite(3, [[0, 1], [1, 2], [2, 0]]);         // false — 길이 3짜리 사이클, 두 집합으로 분할 불가능
isBipartite(5, [[0, 1], [2, 3]]);                 // true  — 연결되지 않은 두 컴포넌트, 각각 분할 가능
isBipartite(1, []);                               // true  — 간선 없음, 정점 하나는 자명하게 이분
isBipartite(1, [[0, 0]]);                         // false — 자기 루프, 분할 불가능
isBipartite(6, [[0, 1], [0, 2], [1, 3], [1, 4], [2, 5]]); // true — 트리는 항상 이분 그래프
```
