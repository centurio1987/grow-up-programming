# 최소 비용 최대 유량 (Min-Cost Max-Flow)

## 한 줄 요약

> 단위 유량당 비용이 있는 유량 네트워크의 정점 수 `n`, 간선 목록 `edges`, 소스 `source`, 싱크 `sink` 를 받아, **최대 유량을 흘리는 모든 방법 중 총 비용이 최소인 값**을 `{ flow, cost }` 형태로 반환한다.

## 스토리

물류 회사가 공장에서 항구까지 제품을 운송한다. 도로마다 한 번에 실을 수 있는 최대 물량(용량)이 다르고, 물량 한 단위를 운반할 때마다 도로 통행료(단위 비용)도 다르다. 회사는 항구로 보낼 수 있는 **최대한 많은 물량**을 보내야 하지만, 그 물량을 보내는 방법이 여러 가지라면 **운송 비용이 가장 적게 드는 경로 조합**을 선택하고 싶다.

운송 계획 팀이 도로망 정보를 넘겨주면, 최대로 보낼 수 있는 물량과 그때의 최소 비용을 함께 알려주는 함수가 필요하다. 최대 물량을 포기하면서 비용을 줄이는 것은 허용되지 않는다.

## 함수 인터페이스

```ts
export function minCostMaxFlow(
  n: number,
  edges: [number, number, number, number][],
  source: number,
  sink: number,
): { flow: number; cost: number };
```

- `n` — 정점의 개수. 정점 번호는 $0 \ldots n-1$.
- `edges` — 방향 간선 목록. 각 원소 `[u, v, c, a]` 는 정점 `u` 에서 `v` 로 향하는 용량 `c`, 단위 비용 `a` 의 간선이다.
- `source` — 소스 정점 번호.
- `sink` — 싱크 정점 번호.
- 반환 — `{ flow, cost }`. `flow` 는 소스에서 싱크로의 최대 유량, `cost` 는 그때의 최소 총 비용.

## 제약 조건

- $2 \leq n \leq 200$
- $0 \leq E \leq 2 \cdot 10^3$ (여기서 $E$ 는 간선의 수)
- $0 \leq c(u, v) \leq 10^4$ (용량)
- $0 \leq a(u, v) \leq 10^4$ (단위 비용)
- $0 \leq \text{source},\, \text{sink} < n,\;\; \text{source} \neq \text{sink}$
- 정점 번호는 $0 \ldots n-1$.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

비용이 있는 유량 네트워크 $G = (V, E, c, a)$ 에서 $c(u, v)$ 는 용량, $a(u, v)$ 는 단위 유량당 비용이다.

유량 함수 $f$ 의 총 비용:

$$\text{cost}(f) = \sum_{(u, v) \in E} a(u, v) \cdot f(u, v)$$

목적: 최대 유량 $|f^*|$ 을 달성하는 유량 함수들 중 총 비용이 가장 작은 것을 구한다.

$$\text{minCostMaxFlow}(G, s, t) = \min_{\substack{f \text{ feasible}\\ |f| = |f^*|}} \text{cost}(f)$$

- 소스에서 싱크로 가는 경로가 없으면 `{ flow: 0, cost: 0 }` 을 반환한다.
- 용량 `0` 인 간선은 사용할 수 없다.
- 비용 `0` 인 간선은 용량만큼 비용 없이 흘릴 수 있다.
- 최대 유량이 동일하다면, 그 중에서 비용이 더 낮은 경로 조합을 선택한다.

## 예시

```ts
minCostMaxFlow(2, [[0, 1, 5, 2]], 0, 1);
// { flow: 5, cost: 10 } — 유량 5를 단위 비용 2짜리 직통 경로로 모두 전달, 총 비용 5×2=10

minCostMaxFlow(3, [[0, 1, 5, 2], [1, 2, 5, 3]], 0, 2);
// { flow: 5, cost: 25 } — 직렬 경로, 단위당 비용 2+3=5, 총 비용 5×5=25

minCostMaxFlow(
  4,
  [[0, 1, 1, 1], [1, 3, 1, 1], [0, 2, 2, 5], [2, 3, 2, 5]],
  0, 3,
);
// { flow: 3, cost: 22 } — 최대 유량 3; 싼 경로(비용 2)로 1단위, 비싼 경로(비용 10)로 2단위

minCostMaxFlow(3, [[0, 1, 10, 5]], 0, 2);
// { flow: 0, cost: 0 } — 싱크 2로 향하는 경로 없음

minCostMaxFlow(2, [[0, 1, 1, 0]], 0, 1);
// { flow: 1, cost: 0 } — 단위 비용 0, 유량 1, 비용 없음

minCostMaxFlow(
  4,
  [[0, 1, 2, 1], [1, 3, 2, 1], [0, 2, 3, 10], [2, 3, 3, 10]],
  0, 3,
);
// { flow: 5, cost: 64 } — 싼 경로(단위 비용 2)로 2단위, 비싼 경로(단위 비용 20)로 3단위
```
