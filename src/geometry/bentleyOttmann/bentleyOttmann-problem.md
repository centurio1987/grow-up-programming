# 여러 선분의 교차 개수

## 한 줄 요약

> 선분 배열이 주어질 때, 서로 교차하는 선분 **쌍**의 개수를 반환한다.

## 스토리

도시 계획부의 하은은 신규 도로 노선도를 검토하고 있다. 여러 도로 노선이 지도 위에 직선 선분으로 표시되어 있고, 두 노선이 만나는 지점마다 교차로를 설치해야 한다.

같은 지점에서 세 개 이상의 노선이 동시에 만나더라도, 관리 시스템에는 "노선 쌍" 단위로 교차로를 등록한다. 두 노선의 끝점이 딱 맞닿거나 한 노선 위에 다른 노선이 겹쳐 올라타도 교차로로 인정된다.

주어진 선분 목록에서 교차하는 쌍의 개수를 구하는 함수를 만들어라.

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function bentleyOttmann(segments: Segment[]): number;
```

- `segments` — 선분 배열. 각 원소는 `[[x1, y1], [x2, y2]]` 형태의 두 끝점.
- 반환 — 교차하는 선분 쌍의 개수. $\geq 0$ 인 정수.

## 제약 조건

- $0 \leq N \leq 10^5$ — 여기서 $N$ 은 선분의 개수. 선분이 0개이면 0을 반환한다.
- $-10^9 \leq x, y \leq 10^9$ — 끝점 좌표 범위
- 반환값은 $\geq 0$ 인 정수
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

$N$ 개의 선분 $S = \{s_1, s_2, \ldots, s_N\}$ 중 서로 교차하는 쌍의 수를 반환한다.

$$\text{count}(S) = \left| \{ (i, j) \mid i < j,\; s_i \cap s_j \neq \emptyset \} \right|$$

교차의 기준은 다음과 같다.

- 두 선분이 내부에서 한 점에서 만나면 교차한다.
- 끝점이 다른 선분의 끝점과 일치하면 교차한다 (공유 끝점).
- 한 선분의 끝점이 다른 선분의 내부에 있으면 교차한다 (T자).
- 두 선분이 같은 직선 위에서 구간을 공유하면 교차한다 (공선 겹침).
- 같은 점에서 $k$ 개의 선분이 동시에 만나면, 쌍의 수는 $\binom{k}{2}$ 로 계산한다.

## 예시

```ts
// X자 교차 — 두 선분이 중앙에서 만남
bentleyOttmann([[[0, 0], [4, 4]], [[0, 4], [4, 0]]]); // 1

// 분리된 두 선분 — 교차 없음
bentleyOttmann([[[0, 0], [1, 0]], [[2, 0], [3, 0]]]); // 0

// 끝점 공유 (L자)
bentleyOttmann([[[0, 0], [1, 0]], [[1, 0], [1, 1]]]); // 1

// 공선 겹침 — 구간 [2,4]를 공유
bentleyOttmann([[[0, 0], [4, 0]], [[2, 0], [6, 0]]]); // 1

// 3개 선분이 한 점에서 만남 — 쌍은 (1,2), (1,3), (2,3) 총 3쌍
bentleyOttmann([[[0, 0], [4, 4]], [[0, 4], [4, 0]], [[2, 0], [2, 4]]]); // 3

// 격자형 — 가로 2개 × 세로 2개 → 4교차
bentleyOttmann([
  [[0, 1], [10, 1]],
  [[0, 3], [10, 3]],
  [[2, 0], [2, 5]],
  [[5, 0], [5, 5]],
]); // 4

// 선분 0개
bentleyOttmann([]); // 0
```
