# 여러 선분의 교차 개수

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 고급 |

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function bentleyOttmann(segments: Segment[]): number;
```

## 제약 조건

- $1 \leq n \leq 10^5$ ($n$ 은 선분의 개수)
- $-10^9 \leq x, y \leq 10^9$
- 반환값은 $\geq 0$ 인 정수

## 문제 상세

$n$ 개의 선분 $S = \{s_1, s_2, \ldots, s_n\}$ 가 주어질 때, 서로 다른 선분 쌍의 교차 횟수를 반환하라. 같은 점에서 여러 선분이 동시에 만나더라도 쌍 단위로 센다:

$$\text{count}(S) = \left| \{ (i, j) \mid i < j,\; s_i \cap s_j \neq \emptyset \} \right|$$

끝점에서 닿는 경우(공유 끝점)도 교차로 셈한다.

## 예시

| `segments` | 반환값 | 설명 |
| --- | --- | --- |
| `[[[0,0],[2,2]], [[0,2],[2,0]]]` | `1` | $X$자 형태로 한 점에서 교차 |
| `[[[0,0],[1,0]], [[2,0],[3,0]]]` | `0` | 서로 떨어진 두 선분 |
| `[[[0,0],[2,0]], [[1,0],[3,0]]]` | `1` | 같은 직선 위에서 겹치는 두 선분 |
| `[[[0,0],[2,0]], [[2,0],[2,2]]]` | `1` | 끝점에서 닿는 두 선분 ($L$자) |
