# 점 집합의 지름 (제곱 거리)

## 한 줄 요약

> 2D 평면 위 점 배열이 주어질 때, 가장 멀리 떨어진 두 점 사이 거리의 **제곱**을 반환한다.

## 스토리

우주 관측소의 데이터 분석팀은 관측 구역 안 별들의 좌표를 가지고 있다. 관측 구역의 "지름"을 구하려고 하는데, 지름이란 가장 멀리 떨어진 두 별 사이의 거리를 뜻한다.

정밀 계산이 필요한 상황에서 부동소수점 오차를 피하기 위해, 거리 자체가 아니라 거리의 제곱을 반환하기로 했다. 별들의 좌표는 정수이므로 제곱 거리도 정수로 딱 떨어진다.

주어진 점 집합에서 가장 멀리 떨어진 두 점 사이 거리의 제곱을 구하는 함수를 만들어라.

## 함수 인터페이스

```ts
export type Point = [number, number];

export function rotatingCalipersDiameter(points: Point[]): number;
```

- `points` — 2D 평면 위 점 배열. 각 원소는 `[x, y]` 형태. 배열의 길이는 $N$.
- 반환 — 가장 멀리 떨어진 두 점 사이의 거리의 제곱. $\geq 0$ 인 수.

## 제약 조건

- $2 \leq N \leq 10^5$ — 점의 개수는 최소 2개
- $-10^9 \leq x, y \leq 10^9$ — 정수 좌표 권장
- 반환값은 $\geq 0$ 이며, 좌표가 정수이면 반환값도 정수
- 최대 반환값은 $(2 \times 10^9)^2 = 4 \times 10^{18}$ 이므로 64비트 부동소수 처리 필요
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

2D 평면 위 $N$ 개의 점 $P = \{p_1, \ldots, p_N\}$ 에서 서로 다른 인덱스 $i \neq j$ 인 두 점 사이의 거리가 최대인 쌍의 거리 제곱을 반환한다.

$$\mathrm{diam}^2(P) = \max_{i \neq j} \left( (x_i - x_j)^2 + (y_i - y_j)^2 \right)$$

- 두 점의 좌표가 동일하면 제곱 거리는 $0$ 이다.
- 반환값은 제곱 거리이므로 루트를 취하지 않는다.
- 좌표 범위가 $\pm 10^9$ 이므로, 두 좌표의 차이가 최대 $2 \times 10^9$ 이고 제곱하면 $4 \times 10^{18}$ 까지 커진다. 오버플로 없이 처리해야 한다.

## 예시

```ts
// 두 점만 있는 경우
rotatingCalipersDiameter([[0, 0], [3, 4]]); // 25 — 3²+4²=25

// 정사각형 — 대각선이 지름
rotatingCalipersDiameter([[0, 0], [1, 0], [1, 1], [0, 1]]); // 2 — √2의 제곱

// 직사각형 — 대각선이 지름
rotatingCalipersDiameter([[0, 0], [4, 0], [4, 6], [0, 6]]); // 52 — 4²+6²=52

// 내부 점이 많아도 외곽 두 점이 답
rotatingCalipersDiameter([[0, 0], [10, 0], [5, 5], [4, 3], [6, 2]]); // 100 — (0,0)과 (10,0)

// 공선 점들 — 양 끝점 사이
rotatingCalipersDiameter([[0, 0], [2, 0], [5, 0], [9, 0]]); // 81 — 9²=81

// 동일 좌표 두 점
rotatingCalipersDiameter([[7, 7], [7, 7]]); // 0 — 같은 점

// 큰 좌표 — 오버플로 없이 처리
rotatingCalipersDiameter([[-1_000_000_000, 0], [1_000_000_000, 0], [0, 0]]); // 4e18 — (2×10⁹)²
```
