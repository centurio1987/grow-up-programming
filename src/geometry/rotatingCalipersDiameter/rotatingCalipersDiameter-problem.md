# 점 집합의 지름 (제곱 거리)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 고급 |

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function rotatingCalipersDiameter(points: Point[]): number;
```

## 제약 조건

- $2 \leq n \leq 10^5$ ($n$ 은 점의 개수)
- $-10^9 \leq x, y \leq 10^9$ (정수 좌표 권장)
- 반환값은 $\geq 0$ 인 수

## 문제 상세

2차원 평면 위 $n$ 개의 점 $P = \{p_1, \ldots, p_n\}$ 의 지름은 가장 멀리 떨어진 두 점 사이의 거리이다. 부동소수 오차를 피하기 위해 **거리는 제곱 형태로 반환한다**:

$$\mathrm{diam}^2(P) = \max_{i \neq j} \left( (x_i - x_j)^2 + (y_i - y_j)^2 \right)$$

## 예시

| `points` | 반환값 | 설명 |
| --- | --- | --- |
| `[[0,0], [3,4]]` | `25` | $3^2 + 4^2 = 25$ |
| `[[0,0], [1,0], [0,1], [1,1]]` | `2` | 대각선 $\sqrt{2}$ 의 제곱 |
| `[[0,0], [10,0], [5,5], [0,10], [10,10]]` | `200` | $(0,0)$ 과 $(10,10)$ 사이 거리의 제곱 |
| `[[0,0], [0,0]]` | `0` | 같은 점 두 개 |
