# 점이 다각형 내부인지 판정

## 한 줄 요약

> 점 `p` 와 단순 다각형 `polygon` 이 주어질 때, `p` 가 다각형의 내부 또는 경계 위에 있으면 `true`, 외부이면 `false` 를 반환한다.

## 스토리

게임 개발팀의 유나는 2D 맵에서 플레이어의 위치가 특정 구역 안에 있는지를 매 프레임마다 판단해야 한다. 구역은 여러 꼭짓점을 직선으로 이어 만든 다각형으로 정의되며, 오목한 형태(L자, 별 모양 등)일 수도 있다.

플레이어가 구역 경계 위에 정확히 올라서 있을 때도 "구역 안"으로 처리해야 한다. 구역을 벗어난 순간만 "외부"로 간주한다.

주어진 점이 다각형 내부 또는 경계에 있는지 판정하는 함수를 만들어라.

## 함수 인터페이스

```ts
export type Point = [number, number];

export function pointInPolygon(p: Point, polygon: Point[]): boolean;
```

- `p` — 판정할 점. `[x, y]` 형태의 2D 좌표.
- `polygon` — 단순 다각형의 꼭짓점 배열. 각 원소는 `[x, y]`. 마지막 꼭짓점과 첫 번째 꼭짓점이 자동으로 연결된다.
- 반환 — `p` 가 다각형 내부이거나 경계 위에 있으면 `true`, 외부이면 `false`.

## 제약 조건

- $3 \leq N \leq 10^5$ — 여기서 $N$ 은 꼭짓점 배열의 길이
- $-10^9 \leq x, y \leq 10^9$ — 좌표 범위
- `polygon` 은 자기교차가 없는 단순 다각형 (볼록·오목 모두 가능)
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

점 $p = (x_0, y_0)$ 가 정점 $v_0, v_1, \ldots, v_{n-1}$ 로 이루어진 단순 다각형의 내부 또는 경계에 있는지 판정한다.

- 점이 다각형의 **변 위**에 있으면 `true` 를 반환한다.
- 점이 다각형의 **정점과 동일한 위치**에 있으면 `true` 를 반환한다.
- 점이 다각형 **내부 영역**에 있으면 `true` 를 반환한다.
- 점이 다각형 외부에 있으면 `false` 를 반환한다.
- 다각형이 오목한 모양이어도 올바른 결과를 반환해야 한다. 오목 부분의 빈 공간은 외부로 처리한다.

## 예시

```ts
const square: Point[] = [[0, 0], [4, 0], [4, 4], [0, 4]];

// 내부
pointInPolygon([2, 2], square); // true — 정사각형 정중앙

// 외부
pointInPolygon([5, 5], square); // false — 오른쪽 위 바깥

// 변 위 — 경계는 내부로 간주
pointInPolygon([2, 0], square); // true — 아랫변 중앙

// 정점 위 — 경계는 내부로 간주
pointInPolygon([0, 0], square); // true — 좌하단 꼭짓점

// 오목 다각형 (L자)
const lShape: Point[] = [[0, 0], [4, 0], [4, 2], [2, 2], [2, 4], [0, 4]];
pointInPolygon([1, 1], lShape); // true — L자 좌하단 내부
pointInPolygon([3, 3], lShape); // false — L자의 오목 부분(파인 공간)은 외부

// 삼각형
const tri: Point[] = [[0, 0], [6, 0], [3, 6]];
pointInPolygon([3, 2], tri); // true — 삼각형 무게중심 근처
pointInPolygon([5, 5], tri); // false — 삼각형 외부 오른쪽
```
