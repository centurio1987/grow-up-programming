# 점이 다각형 내부인지 판정

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function pointInPolygon(p: Point, polygon: Point[]): boolean;
```

## 제약 조건

- $3 \leq n \leq 10^5$ ($n$ 은 다각형 정점의 개수)
- $-10^9 \leq x, y \leq 10^9$
- `polygon` 은 자기교차가 없는 단순 다각형 (볼록·오목 무관)

## 문제 상세

점 $p = (x_0, y_0)$ 가 정점 $v_0, v_1, \ldots, v_{n-1}$ 로 이루어진 단순 다각형 내부에 있는지 판정하라.

- 점이 정확히 변 위 또는 정점 위에 있으면 **내부(true)** 로 간주한다.
- 점이 다각형의 경계 또는 내부 영역에 있으면 `true`, 외부이면 `false` 를 반환한다.

## 예시

| `p` | `polygon` | 반환값 |
| --- | --- | --- |
| `[1, 1]` | `[[0,0], [2,0], [2,2], [0,2]]` | `true` |
| `[3, 3]` | `[[0,0], [2,0], [2,2], [0,2]]` | `false` |
| `[1, 0]` | `[[0,0], [2,0], [2,2], [0,2]]` | `true` (변 위) |
| `[0, 0]` | `[[0,0], [2,0], [2,2], [0,2]]` | `true` (정점) |
| `[1, 1]` | `[[0,0], [4,0], [4,4], [2,2], [0,4]]` | `true` (오목 다각형 내부) |
