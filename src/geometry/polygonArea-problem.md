# 다각형 면적

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function polygonArea(polygon: Point[]): number;
```

## 제약 조건

- $3 \leq n \leq 10^5$ ($n$ 은 정점의 개수)
- $-10^9 \leq x, y \leq 10^9$
- `polygon` 은 자기교차가 없는 단순 다각형
- 반환값은 $\geq 0$ 인 부동소수

## 문제 상세

정점 $v_0 = (x_0, y_0), v_1 = (x_1, y_1), \ldots, v_{n-1} = (x_{n-1}, y_{n-1})$ 로 정의된 단순 다각형의 면적은 다음과 같이 계산된다:

$$A = \frac{1}{2} \left| \sum_{i=0}^{n-1} (x_i\, y_{i+1} - x_{i+1}\, y_i) \right|$$

여기서 인덱스는 순환적이며 $v_n = v_0$ 이다.

**절댓값을 취하므로 정점 순서가 시계 방향이든 반시계 방향이든 동일한 비음수 결과를 반환해야 한다.**

## 예시

| `polygon` | 반환값 | 설명 |
| --- | --- | --- |
| `[[0,0], [2,0], [2,2], [0,2]]` | `4` | $2 \times 2$ 정사각형 |
| `[[0,0], [4,0], [0,3]]` | `6` | 밑변 $4$, 높이 $3$ 인 직각삼각형 |
| `[[0,2], [2,2], [2,0], [0,0]]` | `4` | 시계 방향이어도 동일 |
| `[[0,0], [4,0], [4,4], [2,2], [0,4]]` | `12` | 오목 다각형 |
