# 두 선분 교차 판정

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 중급 |

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function segmentsIntersect(s1: Segment, s2: Segment): boolean;
```

## 제약 조건

- $-10^9 \leq x, y \leq 10^9$
- 각 `Segment` 는 두 끝점 $[p_1, p_2]$ 로 정의된다.

## 문제 상세

두 선분 $s_1 = \overline{p_1 p_2}$, $s_2 = \overline{p_3 p_4}$ 가 교차하는지 여부를 반환하라.

### 교차의 정의

- 두 선분이 한 점(또는 그 이상)을 공유하면 교차한다.
- 두 선분이 공선이고 일부 구간을 공유해도 교차로 간주한다.
- 끝점이 다른 선분 위에 있거나 끝점끼리 닿는 경우도 **교차로 간주한다** ($T$ 자, $L$ 자 포함).

## 예시

| `s1` | `s2` | 반환값 | 설명 |
| --- | --- | --- | --- |
| `[[0,0],[2,2]]` | `[[0,2],[2,0]]` | `true` | $X$ 자 교차 |
| `[[0,0],[1,0]]` | `[[2,0],[3,0]]` | `false` | 같은 직선이지만 떨어져 있음 |
| `[[0,0],[2,0]]` | `[[1,0],[3,0]]` | `true` | 공선 위 일부 구간 공유 |
| `[[0,0],[2,0]]` | `[[2,0],[2,2]]` | `true` | 끝점에서 닿음 ($L$ 자) |
| `[[0,0],[2,2]]` | `[[1,1],[3,0]]` | `true` | 끝점이 다른 선분 위 ($T$ 자) |
| `[[0,0],[1,1]]` | `[[2,2],[3,3]]` | `false` | 같은 직선이지만 떨어져 있음 |
