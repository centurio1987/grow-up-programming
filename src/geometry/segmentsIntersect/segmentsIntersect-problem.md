# 두 선분 교차 판정

## 한 줄 요약

> 두 선분 `s1`, `s2` 가 주어질 때, 두 선분이 한 점이라도 공유하면 `true`, 아니면 `false` 를 반환한다.

## 스토리

도시 교통 시스템 팀이 지도 위에 새 도로 두 개를 검토하고 있다. 두 도로가 교차하는 지점이 있으면 신호등을 설치해야 하고, 교차하지 않으면 설치가 불필요하다.

도로는 각각 시작점과 끝점으로 정의된 직선 선분이다. 두 도로의 끝점이 정확히 맞닿는 경우, 한 도로의 끝이 다른 도로 위에 있는 경우, 두 도로가 일부 구간을 공유하는 경우 모두 "교차 있음"으로 처리해야 한다.

주어진 두 선분이 교차하는지 판정하는 함수를 만들어라.

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function segmentsIntersect(s1: Segment, s2: Segment): boolean;
```

- `s1` — 첫 번째 선분. `[[x1, y1], [x2, y2]]` 형태로 두 끝점을 나타낸다.
- `s2` — 두 번째 선분. 같은 형태로 두 끝점을 나타낸다.
- 반환 — 두 선분이 한 점이라도 공유하면 `true`, 완전히 분리되어 있으면 `false`.

## 제약 조건

- $-10^9 \leq x, y \leq 10^9$ — 각 끝점의 좌표는 이 범위 안의 정수
- 선분은 길이가 0인 점(시작점 = 끝점)도 유효한 입력이다
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

두 선분 $s_1 = \overline{p_1 p_2}$, $s_2 = \overline{p_3 p_4}$ 가 교차하는지 판정한다.

교차의 정의는 다음과 같다.

- 두 선분이 정확히 한 점에서 만나면 교차한다 (X자, T자 포함).
- 한 선분의 끝점이 다른 선분의 내부 또는 끝점에 있으면 교차한다 (L자 포함).
- 두 선분이 같은 직선 위에서 일부 구간을 공유하면 교차한다.
- 두 선분이 같은 직선 위에 있더라도 공유 구간이 없으면 교차하지 않는다.
- 완전히 평행하고 분리되어 있으면 교차하지 않는다.

## 예시

```ts
// X자 교차
segmentsIntersect([[0, 0], [2, 2]], [[0, 2], [2, 0]]); // true — 두 대각선이 중심에서 만남

// 평행하고 분리됨
segmentsIntersect([[0, 0], [2, 0]], [[0, 1], [2, 1]]); // false — y값이 달라 만나지 않음

// 끝점끼리 닿는 경우 (L자)
segmentsIntersect([[0, 0], [1, 0]], [[1, 0], [1, 1]]); // true — (1,0)을 공유

// 한 끝점이 다른 선분 위에 있는 경우 (T자)
segmentsIntersect([[0, 0], [4, 0]], [[2, 0], [2, 3]]); // true — (2,0)이 s1 위에 있음

// 공선이면서 구간이 겹침
segmentsIntersect([[0, 0], [3, 0]], [[2, 0], [5, 0]]); // true — [2,3] 구간 공유

// 공선이지만 분리됨
segmentsIntersect([[0, 0], [1, 0]], [[2, 0], [3, 0]]); // false — 같은 직선이나 공유 점 없음

// 교차할 것 같지만 한쪽이 짧아 닿지 않음
segmentsIntersect([[0, 0], [10, 10]], [[5, 0], [5, 4]]); // false — s1이 x=5일 때 y=5, s2의 y 최대는 4
```
