# 가장 가까운 두 점 쌍

## 함수 인터페이스

```ts
export type Point = [number, number];
export type Segment = [Point, Point];

export function closestPairOfPoints(points: Point[]): number;
```

## 제약 조건

- $2 \leq n \leq 10^5$ ($n$ 은 점의 개수)
- $-10^9 \leq x, y \leq 10^9$
- 반환값은 $\geq 0$ 인 부동소수

## 문제 상세

2차원 평면 위 $n$ 개의 점 $P = \{p_1, \ldots, p_n\}$ 중 유클리드 거리가 최소인 두 점 쌍의 거리를 반환하라:

$$d^{*}(P) = \min_{i \neq j} \sqrt{(x_i - x_j)^2 + (y_i - y_j)^2}$$

같은 좌표의 점이 둘 이상 있을 경우 최솟값은 $0$ 이다.

## 예시

| `points` | 반환값 | 설명 |
| --- | --- | --- |
| `[[0,0], [3,4]]` | `5` | 두 점 사이 거리 |
| `[[0,0], [1,0], [10,10]]` | `1` | 가장 가까운 쌍은 $(0,0)$ 과 $(1,0)$ |
| `[[0,0], [0,0], [5,5]]` | `0` | 동일 좌표 점이 존재 |
| `[[0,0], [2,0], [1,1], [3,3]]` | `\sqrt{2}` | $(0,0)$ 과 $(1,1)$ 또는 $(2,0)$ 과 $(1,1)$ |
