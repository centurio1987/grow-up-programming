# 가장 가까운 두 점 쌍

## 한 줄 요약

> 2D 평면 위 점 배열이 주어질 때, 가장 가까운 두 점 사이의 **유클리드 거리**를 반환한다.

## 스토리

무선 통신망 설계를 맡은 지훈은 도시 전역에 흩어진 기지국 좌표 목록을 가지고 있다. 가장 가까이 붙어 있는 두 기지국을 찾아 주파수 간섭 위험을 점검해야 한다.

수십만 개의 기지국 좌표를 받아 가장 짧은 기지국 간 거리를 빠르게 계산하는 함수가 필요하다.

주어진 점 집합에서 가장 가까운 두 점 사이의 거리를 구하는 함수를 만들어라.

## 함수 인터페이스

```ts
export type Point = [number, number];

export function closestPairOfPoints(points: Point[]): number;
```

- `points` — 2D 평면 위 점 배열. 각 원소는 `[x, y]` 형태. 배열의 길이는 $N$.
- 반환 — 가장 가까운 두 점 사이의 유클리드 거리. $\geq 0$ 인 부동소수.

## 제약 조건

- $2 \leq N \leq 10^5$ — 점의 개수는 최소 2개
- $-10^9 \leq x, y \leq 10^9$ — 좌표 범위
- 동일 좌표를 가진 점이 둘 이상 존재할 수 있으며, 이 경우 최솟값은 $0$ 이다
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

2D 평면 위 $N$ 개의 점 $P = \{p_1, \ldots, p_N\}$ 에서 서로 다른 인덱스 $i \neq j$ 인 두 점 사이의 유클리드 거리가 최소인 값을 반환한다.

$$d^{*}(P) = \min_{i \neq j} \sqrt{(x_i - x_j)^2 + (y_i - y_j)^2}$$

- 동일한 좌표를 가진 점이 둘 이상 있으면 반환값은 $0$ 이다.
- 거리는 유클리드 거리(직선 거리)를 기준으로 한다.
- 반환값은 부동소수이며 정밀도 오차가 허용된다 (상대 오차 $10^{-9}$ 이하 권장).

## 예시

```ts
// 두 점만 있는 경우
closestPairOfPoints([[0, 0], [3, 4]]); // 5 — √(9+16) = 5

// 여러 점 중 가까운 쌍
closestPairOfPoints([[0, 0], [1, 0], [10, 10]]); // 1 — (0,0)과 (1,0) 사이 거리

// 동일 좌표 점 존재
closestPairOfPoints([[0, 0], [0, 0], [5, 5]]); // 0 — 같은 점이 두 개

// x축 방향으로 가장 가까운 쌍
closestPairOfPoints([[0, 0], [100, 0], [200, 0], [201, 0]]); // 1 — (200,0)과 (201,0)

// y축 방향으로 가까운 쌍
closestPairOfPoints([[0, 0], [10, 100], [10, 101], [50, 50]]); // 1 — (10,100)과 (10,101)

// 대각선으로 가까운 쌍
closestPairOfPoints([[0, 0], [3, 3], [3, 4]]); // 1 — (3,3)과 (3,4) 사이 거리

// 분할 경계를 가로지르는 쌍이 정답인 경우
closestPairOfPoints([[0, 0], [10, 0], [11, 0], [21, 0]]); // 1 — (10,0)과 (11,0)
```
