# 역순쌍 개수 세기

## 한 줄 요약

> 배열을 받아 앞 원소가 뒤 원소보다 큰 쌍 $(i, j)$의 개수를 반환한다.

## 스토리

도서관 사서 민준은 반납된 책들이 제자리에 꽂혀있는지 확인하는 일을 맡았다. 책들은 일련번호 순서대로 정렬되어야 하는데, 손님이 책을 뺐다 꽂는 과정에서 순서가 뒤섞인다.

민준은 선반 위 책들을 왼쪽에서 오른쪽으로 훑으며 "이 책보다 앞에 있는데 일련번호가 더 큰 쌍"이 몇 쌍인지 셈으로써 선반이 얼마나 무질서해졌는지를 수치화하려 한다.

이 수치가 0이면 완벽히 정렬된 상태, 클수록 더 뒤섞인 상태를 의미한다. 책 수가 수십만 권에 달해도 빠르게 계산할 수 있어야 한다.

## 함수 인터페이스

```ts
export function countInversions(arr: number[]): number;
```

- `arr` — 정수 배열
- 반환 — $i < j$이고 $arr[i] > arr[j]$를 만족하는 쌍 $(i, j)$의 개수

## 제약 조건

- $0 \leq n \leq 10^5$ (여기서 $n$은 `arr`의 길이)
- `arr`의 원소는 정수 (음수 포함 가능)
- 반환값은 $\geq 0$
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

길이 $n$인 정수 배열 $A$가 주어졌을 때, 다음 조건을 동시에 만족하는 쌍 $(i, j)$의 개수를 반환하라.

$$\text{countInversions}(A) = \left|\, \{ (i, j) \;\mid\; 0 \leq i < j < n,\; A[i] > A[j] \} \,\right|$$

- 같은 값끼리($A[i] = A[j]$)는 역순쌍으로 세지 않는다.
- 빈 배열이나 길이 1인 배열은 역순쌍이 존재하지 않으므로 0을 반환한다.

## 예시

```ts
countInversions([1, 2, 3]);       // 0 — 오름차순으로 역순쌍 없음
countInversions([3, 2, 1]);       // 3 — (3,2), (3,1), (2,1)
countInversions([2, 4, 1, 3, 5]); // 3 — (2,1), (4,1), (4,3)

// 경계 케이스
countInversions([]);              // 0 — 빈 배열
countInversions([42]);            // 0 — 원소 하나
countInversions([2, 2, 2]);       // 0 — 같은 값은 역순쌍이 아님
countInversions([-1, -3, 0, -2]); // 3 — (-1,-3), (-1,-2), (0,-2)
```
