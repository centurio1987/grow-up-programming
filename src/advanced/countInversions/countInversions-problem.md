# 역순쌍 개수 (Count Inversions)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 중급 |

## 함수 인터페이스

```ts
export function countInversions(arr: number[]): number;
```

## 제약 조건

- $1 \leq n \leq 10^5$ (여기서 $n$ 은 `arr` 의 길이)
- `arr` 의 원소는 정수
- 반환값은 $\geq 0$

## 문제 상세

길이 $n$ 인 정수 배열 $A$ 가 주어졌을 때, 다음 조건을 만족하는 쌍 $(i, j)$ 의 개수를 구하라.

$$\text{inv}(A) = \left|\, \{ (i, j) \;\mid\; 0 \leq i < j < n,\; A[i] > A[j] \} \,\right|$$

즉, 앞에 있는 원소가 뒤에 있는 원소보다 큰 경우의 수를 모두 세어 반환한다.

## 예시

| `arr` | 반환값 | 설명 |
| --- | --- | --- |
| `[1, 2, 3]` | `0` | 이미 정렬됨 |
| `[3, 2, 1]` | `3` | $(3,2), (3,1), (2,1)$ |
| `[2, 4, 1, 3, 5]` | `3` | $(2,1), (4,1), (4,3)$ |
| `[5]` | `0` | 단일 원소 |
