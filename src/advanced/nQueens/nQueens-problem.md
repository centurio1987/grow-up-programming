# N-Queens

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★★★ 상 — 필수 |
| 난이도 | 중급 |

## 함수 인터페이스

```ts
export function nQueens(n: number): number;
```

## 제약 조건

- $1 \leq n \leq 12$

## 문제 상세

$n \times n$ 크기의 체스판에 $n$ 개의 퀸을 서로 공격할 수 없도록 배치하는 모든 방법의 수를 반환하라.

두 퀸 $(r_1, c_1)$, $(r_2, c_2)$ 가 서로 공격하지 않을 조건은 다음과 같다.

$$r_1 \neq r_2,\quad c_1 \neq c_2,\quad |r_1 - r_2| \neq |c_1 - c_2|$$

즉, 같은 행·같은 열·같은 대각선상에 다른 퀸이 존재해서는 안 된다.
