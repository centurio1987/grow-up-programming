# 폴라드 로 알고리즘 (Pollard's Rho)

## 함수 인터페이스

```ts
export function pollardRho(n: bigint): bigint;
```

## 제약 조건

- $n \geq 2$ (bigint)
- 합성수 입력이 권장됨
- 평균 시간 복잡도 $O(n^{1/4})$

## 문제 상세

큰 합성수 $n$ 의 **비자명한 약수** (즉, $1$ 도 $n$ 도 아닌 약수)를 확률적으로 찾아 반환한다.

알고리즘으로는 **Pollard's Rho** 를 사용한다.

명세:

- 반환값 $d$ 는 $1 < d \leq n$.
- $n$ 이 짝수인 경우 즉시 $2$ 를 반환해도 된다.
- $n$ 이 소수인 경우 비자명한 약수가 존재하지 않으므로 $n$ 자체를 반환할 수 있다 (이 경우는 호출 측에서 사전 판정하거나, 함수 내부에서 처리한다).

## 예시

```ts
pollardRho(15n);          // 3n 또는 5n
pollardRho(8n);           // 2n
pollardRho(91n);          // 7n 또는 13n   (91 = 7 × 13)
pollardRho(1_000_003n);   // 1_000_003n     (소수 → 자기 자신)
pollardRho(10403n);       // 101n 또는 103n (10403 = 101 × 103)
```
