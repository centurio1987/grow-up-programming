# 최대공약수 (Greatest Common Divisor)

## 한 줄 요약

> 함수는 두 정수 `a`, `b`를 받아 두 수의 최대공약수를 반환한다.

## 스토리

타일 공장에서 가로 `a`cm, 세로 `b`cm 크기의 직사각형 바닥을 정사각형 타일로 빈틈없이 채우려 한다. 사용하는 타일은 모두 같은 크기여야 하고 자르지 않아야 한다. 가능한 가장 큰 정사각형 타일의 한 변 길이가 얼마인지 구해야 한다.

예를 들어 바닥이 12cm × 18cm라면 6cm 타일로 채울 수 있다. 7cm 타일은 12와 18을 나누어 떨어지게 할 수 없으므로 불가능하다.

한 쪽 치수가 0이면 다른 쪽 치수가 곧 답이 된다. 두 치수가 모두 0이면 어떤 크기의 타일도 조건을 만족하므로 관례상 0을 반환한다.

## 함수 인터페이스

```ts
export function gcd(a: bigint, b: bigint): bigint;
```

- `a` — 첫 번째 정수, bigint (음수 가능)
- `b` — 두 번째 정수, bigint (음수 가능)
- 반환 — $\gcd(|a|, |b|)$, 항상 $\geq 0$

## 제약 조건

- $a$, $b$는 임의의 bigint (음수·0·양수 모두 허용)
- 결과는 항상 음이 아닌 값: $\gcd(a, b) \geq 0$
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

두 정수 $a$, $b$의 최대공약수를 반환한다.

- 두 인자 중 한쪽이 $0$이면 나머지의 절댓값을 반환한다.
- $a = b = 0$이면 $0$을 반환한다.
- 음수 입력도 절댓값 기준으로 처리한다.

$$\gcd(a, b) = \gcd(|a|, |b|) \geq 0$$

## 예시

```ts
gcd(12n, 18n);    // 6n   — 12와 18의 공약수 중 최대값
gcd(100n, 75n);   // 25n  — 100 = 4·25, 75 = 3·25
gcd(17n, 5n);     // 1n   — 17과 5는 서로소
gcd(0n, 9n);      // 9n   — 한쪽이 0이면 나머지 절댓값
gcd(9n, 0n);      // 9n   — 순서 무관하게 동일
gcd(0n, 0n);      // 0n   — 둘 다 0
gcd(-12n, 18n);   // 6n   — 음수는 절댓값 처리
gcd(-12n, -18n);  // 6n   — 양쪽 음수도 동일
```
