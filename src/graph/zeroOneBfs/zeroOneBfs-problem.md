# 0-1 BFS (가중치 0 또는 1 그래프의 최단 경로)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 중급 |

## 함수 인터페이스

```ts
export function zeroOneBfs(
  n: number,
  edges: [number, number, number][],
  source: number,
): number[];
```

## 제약 조건

- $1 \leq V \leq 10^{5}$ (여기서 $V = n$, 정점 번호는 $0 \ldots n-1$)
- $0 \leq E \leq 10^{5}$ (여기서 $E$ 는 `edges` 의 길이)
- `edges` 의 각 원소 $[u, v, w]$ 는 $u \to v$ 인 유향 간선이며 가중치 $w \in \{0, 1\}$
- $0 \leq \text{source} < n$

## 문제 상세

모든 간선의 가중치가 $0$ 또는 $1$ 인 그래프에서 한 출발점 $s$ 로부터 모든 정점까지의 **최단 거리(가중치 합)** 를 구한다.

가중치 함수 $w: E \to \{0, 1\}$ 이 주어질 때, 정점 $v$ 까지의 거리 $d(v)$ 는 다음과 같이 정의한다:

$$d(v) = \min_{P \in \mathcal{P}(s, v)} \sum_{e \in P} w(e)$$

여기서 $\mathcal{P}(s, v)$ 는 $s$ 에서 $v$ 로 가는 모든 경로의 집합이다.

도달 불가능한 정점에 대해서는 $-1$ 을 반환한다.

반환값은 길이 $n$ 의 배열이며 인덱스 $i$ 의 값은 $d(i)$ 이다 ($d(s) = 0$).

## 예시

다음과 같은 유향 그래프를 생각하자.

```
    0 --1--> 1 --0--> 2
    |                 ^
    +------0----------+
                      |
    3 --1--> 4 -------+
```

- `n = 5`
- `edges = [[0,1,1], [1,2,0], [0,2,0], [3,4,1], [4,2,0]]`
- `source = 0`

| 정점 $i$ | $d(i)$ |
| --- | --- |
| 0 | 0 |
| 1 | 1 |
| 2 | 0 |
| 3 | -1 |
| 4 | -1 |

반환값은 `[0, 1, 0, -1, -1]`.
