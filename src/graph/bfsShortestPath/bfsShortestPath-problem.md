# BFS Shortest Path (무가중치 최단 경로)

## 한 줄 요약

> 함수는 정점 수 `n`, 무향 간선 목록, 출발 정점 `source`를 받아, 각 정점까지의 최단 간선 수를 담은 배열을 반환한다.

## 스토리

소셜 네트워크 팀의 개발자 지수는 "두 사람 사이에 몇 단계를 거쳐야 연결되는가"를 측정하는 기능을 만들고 싶다. 서비스에서 친구 관계는 양방향이고, 아무 가중치 없이 모두 동등한 '한 번의 연결'이다.

지수는 특정 사용자를 기준으로 모든 다른 사용자까지 최소 몇 번의 친구 관계를 거쳐야 닿을 수 있는지 계산하고 싶다. 아예 연결되지 않은 사람은 분리된 그룹에 있으므로 도달 불가 표시로 -1을 쓰기로 했다.

이 정보를 빠르게 뽑아주는 함수가 있으면 "가장 멀리 있는 친구"나 "연결이 끊긴 사용자 그룹"을 쉽게 찾을 수 있다.

## 함수 인터페이스

```ts
export function bfsShortestPath(
  n: number,
  edges: [number, number][],
  source: number,
): number[];
```

- `n` — 정점 수. 정점 번호는 $0 \ldots n-1$
- `edges` — 무향 간선 목록. 각 원소 `[u, v]`는 정점 `u`와 `v` 사이의 무향 간선
- `source` — 출발 정점 번호 ($0 \leq \text{source} < n$)
- 반환 — 길이 `n`의 배열. 인덱스 `i`의 값은 `source`에서 정점 `i`까지의 최단 간선 수. 도달 불가능한 정점은 `-1`. `source` 자신은 `0`

## 제약 조건

- $1 \leq V \leq 10^{5}$ (여기서 $V = n$)
- $0 \leq E \leq 10^{5}$ (여기서 $E$는 `edges`의 길이)
- $0 \leq \text{source} < n$
- 무방향 그래프
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

무가중치 그래프에서 출발점 $s$로부터 모든 정점까지의 **최단 거리(간선 개수)** 를 구한다.

정점 $v$까지의 거리 $d(v)$는 $s$에서 $v$로 가는 경로 중 간선 수가 가장 적은 경로의 간선 수로 정의된다.

반환값은 길이 $n$의 배열이며 인덱스 $i$의 값은 $d(i)$이다.

경계 동작:
- 도달 불가능한 정점의 값은 `-1`이다.
- `source` 자신의 거리는 `0`이다.
- 자기 루프 `[v, v]`는 최단 거리에 영향을 주지 않는다.
- 그래프에 간선이 없으면 `source` 이외의 모든 정점은 `-1`이다.

## 예시

```ts
// 직선 그래프: 0-1-2-3, source=0
// 각 정점은 source에서 1씩 멀어진다
bfsShortestPath(4, [[0,1],[1,2],[2,3]], 0); // [0, 1, 2, 3]

// source가 중간: 0-1-2-3-4, source=2
// 2에서 왼쪽으로 1,2 / 오른쪽으로 1,2
bfsShortestPath(5, [[0,1],[1,2],[2,3],[3,4]], 2); // [2, 1, 0, 1, 2]

// 도달 불가능: 0-1 | 2-3 (분리된 그래프), source=0
// 0과 1만 연결, 2와 3은 도달 불가
bfsShortestPath(4, [[0,1],[2,3]], 0); // [0, 1, -1, -1]

// 간선 없음, source=1
// source 자신만 0, 나머지 -1
bfsShortestPath(4, [], 1); // [-1, 0, -1, -1]

// V=1, 자기 자신만
bfsShortestPath(1, [], 0); // [0]
```
