# Strongly Connected Components (강한 연결 요소, SCC)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 고급 |

## 함수 인터페이스

```ts
export function stronglyConnectedComponents(
  n: number,
  edges: [number, number][],
): number[][];
```

## 제약 조건

- $1 \leq V \leq 10^{5}$ (여기서 $V = n$, 정점 번호는 $0 \ldots n-1$)
- $0 \leq E \leq 10^{5}$ (여기서 $E$ 는 `edges` 의 길이)
- `edges` 의 각 원소 $[u, v]$ 는 $u \to v$ 인 유향 간선

## 문제 상세

유향 그래프 $G = (V, E)$ 에서 **강한 연결 요소(Strongly Connected Component, SCC)** 를 모두 찾는다.

두 정점 $u, v$ 가 같은 SCC 에 속한다는 것은 $u$ 에서 $v$ 로 가는 경로와 $v$ 에서 $u$ 로 가는 경로가 모두 존재한다는 것과 동치이다:

$$u \equiv v \iff (u \to^{*} v) \land (v \to^{*} u)$$

이는 $V$ 위에서의 동치 관계이며, 각 동치류가 하나의 SCC 를 이룬다.

반환 형식:

- 각 SCC 내부의 정점들은 오름차순 정렬한다.
- SCC 자체는 각 SCC 의 첫 원소(즉, 최소 정점 번호) 기준으로 오름차순 정렬한다.

(테스트에서는 SCC 분할이 동일한지를 검증 함수로 확인한다.)

## 예시

다음과 같은 유향 그래프를 생각하자.

```
    0 -> 1 -> 2 -> 0
              |
              v
              3 -> 4
              ^    |
              |____|
```

- `n = 5`
- `edges = [[0,1], [1,2], [2,0], [2,3], [3,4], [4,3]]`

SCC 는 $\{0, 1, 2\}$, $\{3, 4\}$ 의 두 개이다.

반환값은 `[[0, 1, 2], [3, 4]]`.
