# Undirected Cycle Detection (무향 그래프 사이클 탐지)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★★ 중 — 빈출 |
| 난이도 | 중급 |

## 함수 인터페이스

```ts
export function undirectedCycleDetection(
  n: number,
  edges: [number, number][],
): boolean;
```

## 제약 조건

- $1 \leq V \leq 10^{5}$ (여기서 $V = n$, 정점 번호는 $0 \ldots n-1$)
- $0 \leq E \leq 10^{5}$ (여기서 $E$ 는 `edges` 의 길이)
- `edges` 는 무향 간선 목록

## 문제 상세

무향 그래프 $G = (V, E)$ 에 **사이클이 존재하는지 여부** 를 판별한다.

사이클은 서로 다른 정점들로 구성된 닫힌 경로이다:

$$v_{0} \to v_{1} \to \cdots \to v_{k} = v_{0}, \quad k \geq 3$$

추가 규칙:

- 자기 루프 $(v, v)$ 도 사이클로 간주한다.
- 같은 정점 쌍에 대한 다중 간선 $(u, v)$ 가 두 번 이상 등장하면 사이클로 간주한다.

사이클이 존재하면 `true`, 존재하지 않으면 `false` 를 반환한다.

## 예시

| `n` | `edges` | 반환값 | 설명 |
| --- | --- | --- | --- |
| 3 | `[[0,1], [1,2], [2,0]]` | `true` | 삼각형 |
| 4 | `[[0,1], [1,2], [2,3]]` | `false` | 선형 트리 |
| 5 | `[[0,1], [1,2], [3,4]]` | `false` | 두 개의 트리 |
| 1 | `[[0,0]]` | `true` | 자기 루프 |
| 2 | `[[0,1], [0,1]]` | `true` | 다중 간선 |
