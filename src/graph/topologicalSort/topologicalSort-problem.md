# Topological Sort (위상 정렬)

## 한 줄 요약

> 함수는 정점 수 `n`과 유향 간선 목록을 받아, 모든 간선의 방향을 존중하는 정점 순열을 반환하거나, 사이클이 있어 불가능하면 `null`을 반환한다.

## 스토리

교육 플랫폼을 운영하는 팀이 강의 커리큘럼을 설계하고 있다. 어떤 강의는 선수 과목이 있어서, 수강 전에 반드시 먼저 들어야 하는 강의가 정해져 있다. 예를 들어 "알고리즘 심화"를 듣기 전에 "자료구조 기초"를 먼저 수강해야 한다.

팀은 학생들이 규칙을 어기지 않고 모든 강의를 들을 수 있는 수강 순서를 자동으로 생성하고 싶다. 단, 설계 실수로 "강의 A → 강의 B → 강의 A"처럼 서로가 서로의 선수 과목이 되는 순환 관계가 만들어지면 어떤 순서로도 수강이 불가능하다.

이 경우 플랫폼은 "수강 순서를 만들 수 없다"는 신호를 내보내야 한다.

## 함수 인터페이스

```ts
export function topologicalSort(
  n: number,
  edges: [number, number][],
): number[] | null;
```

- `n` — 정점 수. 정점 번호는 $0 \ldots n-1$
- `edges` — 유향 간선 목록. 각 원소 `[u, v]`는 "$u$가 $v$보다 먼저여야 한다"는 $u \to v$ 방향의 제약
- 반환 — 유효한 위상 순서인 길이 `n`의 정점 순열. 사이클이 있으면 `null`. 유효한 순서가 여러 개이면 그 중 어느 것을 반환해도 된다

## 제약 조건

- $1 \leq V \leq 10^{5}$ (여기서 $V = n$)
- $0 \leq E \leq 10^{5}$ (여기서 $E$는 `edges`의 길이)
- 방향 그래프
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

유향 그래프의 정점들을 모든 간선 $(u, v)$에 대해 $u$가 $v$보다 앞에 오도록 일렬로 나열한다.

반환하는 순열 $[v_0, v_1, \ldots, v_{n-1}]$은 다음 조건을 만족해야 한다: 모든 간선 $(u, v)$에 대해 $u$의 위치가 $v$의 위치보다 앞이어야 한다.

경계 동작:
- 사이클이 하나라도 있으면 `null`을 반환한다.
- 자기 루프 `[v, v]`도 사이클로 간주해 `null`을 반환한다.
- 간선이 없으면 임의의 순열이 유효하다.
- 유효한 순열이 여러 개 존재할 수 있으므로 테스트는 순열의 유효성만 검사한다.

## 예시

```ts
// 선형 체인: 0→1→2→3 — 유일한 유효 순서
topologicalSort(4, [[0,1],[1,2],[2,3]]); // [0,1,2,3]

// 다이아몬드 DAG: 0→1, 0→2, 1→3, 2→3
// [0,1,2,3]과 [0,2,1,3] 모두 유효
topologicalSort(4, [[0,1],[0,2],[1,3],[2,3]]); // 유효한 순열 중 하나

// 사이클 포함: 0→1→2→0
topologicalSort(3, [[0,1],[1,2],[2,0]]); // null

// 자기 루프: 0→0
topologicalSort(1, [[0,0]]); // null

// 간선 없음 — 모든 순열 유효
topologicalSort(4, []); // 유효한 순열 중 하나

// V=1, 간선 없음
topologicalSort(1, []); // [0]
```
