# Connected Components (연결 성분)

## 한 줄 요약

> 함수는 정점 수 `n`과 무향 간선 목록을 받아, 서로 연결된 정점 그룹을 정렬된 배열의 배열로 반환한다.

## 스토리

생태학 연구소의 데이터 분석가 하은은 야생동물 이동 패턴을 분석하고 있다. GPS 발신기를 달아 여러 지점을 이동하는 동물들을 추적한 결과, 어떤 지점들은 서로 이동 경로로 연결되어 있고 어떤 지점들은 완전히 고립된 서식지에 있다.

하은은 "같은 생태 구역에 속하는 지점들"을 묶어야 하는데, 두 지점이 같은 구역이라는 것은 직접 이동하거나 다른 지점들을 거쳐 이동할 수 있다는 의미이다. 완전히 고립된 지점은 그 지점 하나만의 독립 구역이다.

이 그룹핑 결과를 구하면 각 생태 구역별로 개체 수와 이동 범위를 따로 분석할 수 있다.

## 함수 인터페이스

```ts
export function connectedComponents(
  n: number,
  edges: [number, number][],
): number[][];
```

- `n` — 정점 수. 정점 번호는 $0 \ldots n-1$
- `edges` — 무향 간선 목록. 각 원소 `[u, v]`는 정점 `u`와 `v` 사이의 무향 간선
- 반환 — 연결 성분의 배열. 각 성분 내 정점들은 오름차순 정렬. 성분들은 각 성분의 최솟값 기준 오름차순 정렬

## 제약 조건

- $1 \leq V \leq 10^{5}$ (여기서 $V = n$)
- $0 \leq E \leq 10^{5}$ (여기서 $E$는 `edges`의 길이)
- 무방향 그래프
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

무향 그래프에서 서로 연결된 정점들의 집합인 **연결 성분(connected component)** 을 모두 찾는다.

두 정점 $u, v$가 같은 연결 성분에 속한다는 것은, $u$와 $v$를 연결하는 경로(직접 또는 중간 정점을 거친 경로)가 존재한다는 것과 동치이다.

반환 형식:
- 각 성분 내부의 정점들은 오름차순 정렬한다.
- 성분 자체는 각 성분의 최소 정점 번호 기준으로 오름차순 정렬한다.

경계 동작:
- 간선이 없으면 모든 정점이 각각 크기 1의 독립 성분이다.
- 자기 루프 `[v, v]`는 연결 관계에 영향을 주지 않는다 — `v`는 여전히 자기 혼자인 성분이다.
- 중복 간선이 있어도 결과는 같다.

## 예시

```ts
// 두 그룹: {0,1,2}와 {3,4}
connectedComponents(5, [[0,1],[1,2],[3,4]]); // [[0,1,2],[3,4]]

// 모두 연결된 사이클: 0-1-2-3-0
connectedComponents(4, [[0,1],[1,2],[2,3],[3,0]]); // [[0,1,2,3]]

// 간선 없음 — 각 정점이 독립 성분
connectedComponents(4, []); // [[0],[1],[2],[3]]

// 자기 루프만 있어도 각각 독립
connectedComponents(3, [[1,1]]); // [[0],[1],[2]]

// V=1 최소 입력
connectedComponents(1, []); // [[0]]
```
