# 공통 조상 찾기

## 한 줄 요약

> 루트가 있는 트리에서 쿼리로 주어진 두 정점 쌍에 대해 각각의 최소 공통 조상 정점 번호를 배열로 반환한다.

## 스토리

오래된 귀족 가문의 족보 기록관은 방대한 가계도를 보관하고 있다. 가계도는 단 한 명의 시조를 뿌리로 삼으며, 모든 구성원은 시조로부터 유일한 혈통 경로로 연결된다.

방문객들은 종종 두 사람을 지목하며 "이 두 사람의 가장 가까운 공통 선조가 누구입니까?"라고 묻는다. 두 사람 모두의 조상이면서 세대가 가장 낮은, 즉 시조로부터 가장 먼 인물을 찾아야 한다. 단, 본인도 자기 자신의 조상으로 간주한다.

질문이 한 번에 수백, 수천 건씩 밀려들기도 한다. 가족 수가 수만 명에 이르는 대가문에서도 즉각 답할 수 있는 방법이 필요하다.

## 함수 인터페이스

```ts
export function lowestCommonAncestor(
  n: number,
  edges: [number, number][],
  root: number,
  queries: [number, number][]
): number[];
```

- `n` — 정점 수
- `edges` — 무방향 간선 목록. 각 원소 `[u, v]`는 정점 `u`와 `v`를 잇는 간선 (0-based 인덱스)
- `root` — 루트 정점 번호 (0-based)
- `queries` — 쿼리 목록. 각 원소 `[u, v]`에 대해 최소 공통 조상을 구한다
- 반환 — 길이 `queries.length`인 배열. 인덱스 `i`에는 `queries[i]`의 최소 공통 조상 정점 번호

## 제약 조건

- $1 \leq N \leq 10^5$ (정점 수)
- `edges`의 길이는 $N - 1$, 주어진 그래프는 연결된 트리를 형성한다
- 각 간선 $[u, v]$에 대해 $0 \leq u, v < N$, $u \neq v$ (0-based)
- $0 \leq \text{root} < N$
- 각 쿼리 $[u, v]$에 대해 $0 \leq u, v < N$
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

두 정점 $u$, $v$의 최소 공통 조상이란 $u$와 $v$ 모두의 조상 중 루트로부터 가장 멀리 떨어진 정점이다.

정점 $x$는 자기 자신의 조상이기도 하다. 따라서 $v$가 $u$의 직접 조상이면 `lca(u, v) == v`이고, `lca(u, u) == u`이다.

`queries`가 비어 있으면 빈 배열을 반환한다.

## 예시

```ts
//        0
//       / \
//      1   2
//     / \   \
//    3   4   5

const edges = [[0,1],[0,2],[1,3],[1,4],[2,5]];

lowestCommonAncestor(6, edges, 0, [[3,4]]);    // [1] — 3과 4의 공통 부모는 1
lowestCommonAncestor(6, edges, 0, [[3,5]]);    // [0] — 3과 5가 처음 만나는 곳은 루트 0
lowestCommonAncestor(6, edges, 0, [[4,2]]);    // [0] — 4와 2의 공통 조상은 0
lowestCommonAncestor(6, edges, 0, [[5,5]]);    // [5] — 같은 정점: 자기 자신
lowestCommonAncestor(6, edges, 0, [[3,1]]);    // [1] — 3의 직접 조상이 1

// 여러 쿼리를 한 번에
lowestCommonAncestor(
  6, edges, 0,
  [[3,4],[3,5],[4,2],[5,5]]
); // [1, 0, 0, 5]

// 경계: 단일 정점 트리
lowestCommonAncestor(1, [], 0, [[0,0]]); // [0]

// 경계: 빈 쿼리 목록
lowestCommonAncestor(6, edges, 0, []);   // []

// 루트가 0이 아닌 경우: 체인 0-1-2-3, root=2
lowestCommonAncestor(4, [[0,1],[1,2],[2,3]], 2, [[0,3],[0,1]]); // [2, 1]
```
