# 트리 지름

## 한 줄 요약

> 가중치 있는 트리에서 두 정점 사이 경로 길이의 최댓값(지름)을 반환한다.

## 스토리

산림청은 국립공원 내 탐방로 네트워크를 관리한다. 탐방로는 분기는 있지만 사이클이 없는 구조로, 어떤 두 지점 사이에도 정확히 하나의 경로만 존재한다. 각 탐방로 구간에는 거리(미터)가 기록되어 있다.

공원 관리자는 "공원에서 가장 멀리 떨어진 두 지점 사이의 거리"를 구하고자 한다. 이 거리를 알면 비상 구조 시 최대 이동 시간을 추산할 수 있다. 경로 길이는 경유하는 모든 구간 거리의 합이다.

지점 수가 수십만 개에 이르더라도 빠르게 답을 구해야 한다.

## 함수 인터페이스

```ts
export function treeDiameter(n: number, edges: [number, number, number][]): number;
```

- `n` — 정점 수
- `edges` — 가중치 있는 무방향 간선 목록. 각 원소 `[u, v, w]`는 정점 `u`와 `v`를 잇고 거리가 `w`인 간선 (0-based 인덱스)
- 반환 — 두 정점 사이 경로 길이의 최댓값 (트리 지름)

## 제약 조건

- $1 \leq N \leq 10^5$ (정점 수)
- `edges`의 길이는 $N - 1$, 주어진 그래프는 연결된 트리를 형성한다
- 각 간선 $[u, v, w]$에 대해 $0 \leq u, v < N$, $u \neq v$, $w \geq 0$ (0-based)
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

두 정점 $u$, $v$ 사이의 경로 길이는 그 경로 위 모든 간선 가중치의 합이다.

트리의 지름은 모든 정점 쌍 중 경로 길이가 가장 큰 값이다.

정점이 하나뿐이면 간선이 없으므로 지름은 0이다.

가중치가 0인 간선도 허용되며, 모든 간선의 가중치가 0이면 지름은 0이다.

## 예시

```ts
treeDiameter(1, []);
// 0 — 정점이 하나: 경로 없음

treeDiameter(2, [[0, 1, 7]]);
// 7 — 유일한 간선 길이

treeDiameter(5, [
  [0, 1, 1],
  [0, 2, 3],
  [1, 3, 2],
  [2, 4, 5],
]);
// 11 — 경로 3→1→0→2→4: 2+1+3+5

treeDiameter(5, [
  [0, 1, 1],
  [0, 2, 2],
  [0, 3, 3],
  [0, 4, 4],
]);
// 7 — 스타 트리: 잎 4에서 잎 3까지 4+3

treeDiameter(5, [
  [0, 1, 2],
  [1, 2, 2],
  [2, 3, 2],
  [3, 4, 2],
]);
// 8 — 체인 트리: 0에서 4까지 2+2+2+2

// 경계: 가중치 0
treeDiameter(3, [[0,1,0],[1,2,0]]);
// 0 — 모든 간선 가중치 0
```
