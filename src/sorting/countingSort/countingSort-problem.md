# Counting Sort

## 한 줄 요약

> $0$ 이상 $1{,}000$ 이하의 정수로만 이루어진 배열 `A`를 받아 **오름차순으로 정렬된 새 배열**을 반환한다.

## 스토리

초등학교 성적 처리실. 오늘 전교생 **`N`명**의 100점 만점 시험 점수가 무작위 순서로 들어왔다.
교사는 점수별 학생 수를 집계해 성적 분포표를 만들고, 낮은 점수부터 순서대로 인쇄물에 찍어야 한다.

점수는 항상 $0$에서 $1{,}000$ 사이의 정수이고, 같은 점수의 학생이 여럿일 수 있다.

## 함수 인터페이스

```ts
export function countingSort(A: number[]): number[];
```

- `A` — 정렬할 비음 정수 배열 (각 원소 $0 \leq A[i] \leq 1{,}000$)
- 반환 — `A`의 모든 원소를 오름차순으로 정렬한 새 배열

## 제약 조건

- $1 \leq N \leq 100{,}000$ — 여기서 $N$은 배열 `A`의 길이다.
- $0 \leq A[i] \leq 1{,}000$ — 각 원소는 이 범위의 **비음 정수**다. 음수나 소수는 입력으로 주어지지 않는다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

배열 `A`를 **오름차순(작은 값 → 큰 값)** 으로 정렬한 결과 배열을 반환한다. 다음 규칙을 따른다.

- **값 범위.** 모든 원소가 $[0, 1{,}000]$ 정수 범위 안에 있다는 조건을 활용해야 한다.
- **원소 집합 보존.** 반환 배열의 다중집합(multiset)은 입력 `A`의 다중집합과 같아야 한다.
- **빈 배열.** `A`가 비어 있으면 빈 배열을 반환한다.
- **중복 허용.** 같은 값이 여럿이면 그 개수 전부 결과에 포함된다.
- 키 범위 $K = 1{,}001$이 고정이므로 반환 배열의 크기는 항상 $N$이다.

## 예시

```ts
countingSort([4, 2, 2, 8, 3, 3, 1]);    // [1, 2, 2, 3, 3, 4, 8] — 중복은 개수 그대로 보존
countingSort([3, 1, 2]);                // [1, 2, 3] — 기본 정렬
countingSort([5]);                      // [5] — 원소가 하나(경계: N=1)
countingSort([7, 7, 7]);               // [7, 7, 7] — 전부 동일(경계: 모든 값 같음)
countingSort([0, 1000, 500, 0, 1000]); // [0, 0, 500, 1000, 1000] — 키 최솟값·최댓값 포함(경계)
countingSort([]);                      // [] — 빈 배열(경계: N=0)
```
