# Quick Sort

## 한 줄 요약

> 정수 배열 `nums`를 받아 **오름차순으로 정렬된 새 배열**을 반환한다.

## 스토리

물류 창고 시스템. 오늘 입고된 **`N`개 택배 상자**의 무게(g)가 무작위 순서로 배열 하나에 담겨 들어왔다.
무게별로 선반을 정리해야 하는 작업자가 말한다. "가장 가벼운 것부터 차례로 줄 세워 주세요."

같은 무게의 상자가 여럿이어도 개수는 그대로 보존되어야 한다.
빈 배열이 들어오면 빈 배열을 돌려주면 된다.

## 함수 인터페이스

```ts
export function quickSort(nums: number[]): number[];
```

- `nums` — 정렬할 정수 배열
- 반환 — `nums`의 모든 원소를 오름차순으로 정렬한 배열

## 제약 조건

- $1 \leq N \leq 50{,}000$ — 여기서 $N$은 배열 `nums`의 길이다.
- $-50{,}000 \leq \text{nums}[i] \leq 50{,}000$ — 각 원소는 이 범위의 정수이며 음수일 수 있다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

배열 `nums`를 **오름차순(작은 값 → 큰 값)** 으로 정렬한 결과를 반환한다. 다음 규칙을 따른다.

- **결과 배열은 오름차순.** $\text{result}[0] \leq \text{result}[1] \leq \cdots \leq \text{result}[N-1]$을 만족해야 한다.
- **원소 집합 보존.** 반환 배열의 다중집합(multiset)은 입력 `nums`의 다중집합과 같아야 한다.
- **빈 배열 입력.** `nums`가 빈 배열이면 빈 배열을 반환한다.
- **중복 허용.** 같은 값이 여럿이면 그 개수 전부 결과에 포함된다.
- 함수가 입력 `nums`를 제자리에서 변경해도 된다.

## 예시

```ts
quickSort([5, 2, 3, 1]);          // [1, 2, 3, 5] — 작은 순으로 나열
quickSort([5, 1, 1, 2, 0, 0]);    // [0, 0, 1, 1, 2, 5] — 중복은 개수 그대로 보존
quickSort([-10, 5, -3, 0, 8]);    // [-10, -3, 0, 5, 8] — 음수 포함 시 음수가 앞에
quickSort([1]);                   // [1] — 원소가 하나(경계: N=1)
quickSort([3, 3, 3]);             // [3, 3, 3] — 전부 동일(경계: 모든 값 같음)
quickSort([5, 4, 3, 2, 1]);       // [1, 2, 3, 4, 5] — 역순 입력
```
