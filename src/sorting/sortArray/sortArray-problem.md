# Sort Array

## 한 줄 요약

> 정수 배열 `A`를 받아 **오름차순으로 정렬된 새 배열**을 반환한다.

## 스토리

도서관 사서 소피아. 오늘 기증받은 **`N`권 책**의 분류 번호가 뒤죽박죽 섞인 채 목록으로 넘어왔다.
이용자들이 책을 찾기 쉽도록 번호 순서로 다시 줄을 세워야 한다.

음수 분류 번호(희귀 특수 컬렉션)도 있고, 같은 번호가 찍힌 책이 여러 권일 수도 있다.
빈 목록이 넘어오면 빈 목록을 돌려준다.

## 함수 인터페이스

```ts
export function sortArray(A: number[]): number[];
```

- `A` — 정렬할 정수 배열
- 반환 — `A`의 모든 원소를 오름차순으로 정렬한 새 배열

## 제약 조건

- $1 \leq N \leq 100{,}000$ — 여기서 $N$은 배열 `A`의 길이다.
- $-10^9 \leq A[i] \leq 10^9$ — 각 원소는 이 범위의 정수이며 음수일 수 있다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

배열 `A`를 **오름차순(작은 값 → 큰 값)** 으로 정렬한 결과 배열 $B$를 반환한다. 결과는 다음을 만족해야 한다.

$$B[0] \leq B[1] \leq \cdots \leq B[N-1], \quad \text{multiset}(B) = \text{multiset}(A)$$

- **빈 배열.** `A`가 비어 있으면 빈 배열을 반환한다.
- **중복 허용.** 같은 값이 여럿이면 그 개수 전부 결과에 포함된다.
- **음수 포함.** 가장 작은 음수가 결과 배열의 첫 번째 자리에 온다.
- **원본 불변.** 반환 배열은 `A`와 다른 새 배열이어야 한다 — 함수가 입력 `A`를 직접 수정해서는 안 된다.

## 예시

```ts
sortArray([5, 2, 3, 1]);          // [1, 2, 3, 5] — 작은 순으로 나열
sortArray([5, 1, 1, 2, 0, 0]);    // [0, 0, 1, 1, 2, 5] — 중복은 개수 그대로 보존
sortArray([-2, 3, -5, 0, 1]);     // [-5, -2, 0, 1, 3] — 음수 포함 시 음수가 앞에
sortArray([42]);                  // [42] — 원소가 하나(경계: N=1)
sortArray([7, 7, 7]);             // [7, 7, 7] — 전부 동일(경계: 모든 값 같음)
sortArray([]);                    // [] — 빈 배열(경계: N=0)
sortArray([1_000_000_000, -1_000_000_000, 0]); // [-1000000000, 0, 1000000000] — 값 범위 경계
```
