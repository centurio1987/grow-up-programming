# Radix Sort

## 함수 인터페이스

```ts
export function radixSort(A: number[]): number[];
```

## 제약 조건

- $1 \leq N \leq 100{,}000$ (여기서 $N$ 은 `A` 의 길이)
- $0 \leq A[i] \leq 10^9$ (비음 정수)

## 문제 상세

$N$ 개의 비음 정수로 이루어진 배열 $A$ 가 주어진다. **기수 정렬(Radix Sort)** 알고리즘을 사용하여 $A$ 를 오름차순으로 정렬한 새 배열을 반환하라.

## 예시

```ts
radixSort([]);                              // []
radixSort([7]);                             // [7]
radixSort([170, 45, 75, 90, 802, 24, 2, 66]);
// [2, 24, 45, 66, 75, 90, 170, 802]
radixSort([0, 1000000000, 1, 999999999]);
// [0, 1, 999999999, 1000000000]
```
