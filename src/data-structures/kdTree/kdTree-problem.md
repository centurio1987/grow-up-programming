# KDTree (K-D 트리)

## 한 줄 요약
> 2차원 공간을 축 교대 방식으로 이분할하여, 최근접 이웃과 범위 검색을 O(log n)에 수행하는 자료구조.

## 스토리

지도 앱에서 "내 주변 카페 찾기" 기능을 구현한다고 생각해 보세요. 카페가 전국에 수백만 개 있다면, 단순히 모든 카페와의 거리를 계산하는 방법은 O(n)이 걸려 현실적으로 불가능합니다.

K-D 트리는 이 문제를 해결합니다. 2차원 지도를 가로(x축)와 세로(y축)로 번갈아 가며 재귀적으로 이분할합니다. 짝수 레벨에서는 x좌표 중앙값 기준으로 왼쪽/오른쪽으로 나누고, 홀수 레벨에서는 y좌표 중앙값 기준으로 위/아래로 나눕니다. 이 구조 덕분에 내 위치와 멀리 떨어진 공간 전체를 가지치기(pruning)하여 탐색 범위를 O(log n)으로 줄일 수 있습니다.

또한 "현재 화면에 보이는 영역 내 카페 모두 표시" 같은 직사각형 범위 검색도 O(√n + k)에 수행할 수 있어, 지도 앱의 줌인/줌아웃 기능에도 효율적으로 활용됩니다.

## 함수 인터페이스

```ts
type Point2D = [number, number];

export class KDTree {
  constructor(points?: Point2D[])
  // 점 집합으로 초기 균형 트리 구성 (중앙값 분할)

  insert(point: Point2D): void
  // 새 점 삽입 — O(log n) 평균

  nearestNeighbor(query: Point2D): Point2D | undefined
  // 가장 가까운 점 반환 (유클리드 거리 기준) — O(log n) 평균

  rangeSearch(min: Point2D, max: Point2D): Point2D[]
  // [min[0], max[0]] × [min[1], max[1]] 직사각형 내 모든 점 — O(√n + k)

  size(): number
  // 저장된 점의 수 — O(1)
}
```

## 제약 조건

- $n \leq 10^4$ (점의 수)
- 좌표 범위: $-10^6 \leq x, y \leq 10^6$
- 시간 제한: 1초, 메모리 제한: 256 MB
- `min[0] ≤ max[0]`, `min[1] ≤ max[1]` (올바른 범위 보장)

## 문제 상세

### 공간 분할 방식

K-D 트리는 K차원 공간을 분할하는 BST(이진 탐색 트리)입니다. 이 문제에서는 K=2로 고정합니다.

**분할 방법:**
- 깊이(depth)가 짝수인 노드는 **x축** 기준으로 점들을 분할합니다.
- 깊이가 홀수인 노드는 **y축** 기준으로 분할합니다.
- 각 노드는 해당 레벨의 분할 축 기준으로 **중앙값** 점이 됩니다.

**초기 균형 구성 (constructor):**
```
1. 주어진 points를 현재 축 기준으로 정렬
2. 중앙값 인덱스의 점을 현재 노드로 선택
3. 왼쪽 절반으로 왼쪽 서브트리, 오른쪽 절반으로 오른쪽 서브트리를 재귀 구성
```

### 최근접 이웃 탐색 (nearestNeighbor)

1. 현재 노드의 점과의 거리를 계산하여 최선 후보를 갱신합니다.
2. 쿼리 점이 현재 노드의 분할 기준보다 작으면 왼쪽 서브트리를 먼저 탐색합니다.
3. 반대쪽 서브트리도 "분할선까지의 거리 < 현재 최선 거리"이면 탐색합니다 (pruning).

### 범위 검색 (rangeSearch)

1. 현재 노드의 점이 범위 안에 있으면 결과에 추가합니다.
2. 범위가 현재 노드의 분할 기준보다 왼쪽(작은) 영역과 겹치면 왼쪽 서브트리를 탐색합니다.
3. 오른쪽 영역과 겹치면 오른쪽 서브트리를 탐색합니다.

## 예시

```ts
const tree = new KDTree([
  [2, 3], [5, 4], [9, 6], [4, 7], [8, 1], [7, 2]
]);

// 최근접 이웃 탐색
tree.nearestNeighbor([9, 2]); // → [8, 1] (유클리드 거리 √2)

// 범위 검색
tree.rangeSearch([4, 1], [9, 5]);
// → [[5, 4], [8, 1], [7, 2]] (순서 무관)

// insert 후
tree.insert([6, 3]);
tree.size(); // → 7
tree.nearestNeighbor([6, 3]); // → [6, 3]
```
