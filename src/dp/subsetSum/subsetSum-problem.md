# 부분집합 합 판정

## 한 줄 요약

> 함수는 비음의 정수 배열과 목표 합을 받아, 원소의 부분집합 중 합이 정확히 목표와 같은 것이 존재하는지 반환한다.

## 스토리

물류 창고 시스템을 개발하는 지훈은 박스 무게 목록을 보고 특정 총 무게를 딱 맞추어 선적할 수 있는지 확인하는 기능을 구현 중이다. 같은 박스를 두 번 실을 수는 없고, 일부 박스만 골라 싣는 것은 허용된다.

목표 무게를 정확히 맞추지 못하면 화물이 반송될 수 있으므로 가능 여부를 미리 판단해야 한다. 박스가 최대 1000개에 달해 모든 조합을 일일이 시도하는 방식은 현실적이지 않다.

## 함수 인터페이스

```ts
export function subsetSum(nums: number[], target: number): boolean;
```

- `nums` — 비음의 정수 배열 (각 원소는 한 번씩만 사용 가능)
- `target` — 만들어야 하는 목표 합
- 반환 — 합이 정확히 `target`인 부분집합이 존재하면 `true`, 없으면 `false`

## 제약 조건

- $1 \leq |nums| \leq 1000$
- $0 \leq nums[i] \leq 10^4$
- $0 \leq target \leq 10^4$
- 각 원소는 0번 또는 1번만 사용할 수 있다.
- 시간 제한: 1초, 메모리 제한: 256 MB

## 문제 상세

빈 부분집합의 합은 $0$이다. 따라서 `target`이 $0$이면 항상 `true`를 반환한다.

`nums`가 비어 있고 `target`이 $0$이면 `true`, $0$보다 크면 `false`를 반환한다.

원소 중에 $0$이 포함되어 있어도 규칙은 동일하게 적용된다.

## 예시

```ts
subsetSum([1, 2, 3], 5);
// true — 2 + 3 = 5

subsetSum([1, 2, 3], 7);
// false — 최대 합이 6이므로 7은 불가능

subsetSum([3, 34, 4, 12, 5, 2], 9);
// true — 4 + 5 = 9

subsetSum([1, 2, 3], 0);
// true — 빈 부분집합의 합은 0

subsetSum([5], 4);
// false — 유일한 원소가 4가 아님

subsetSum([], 0);
// true — 빈 배열의 빈 부분집합 합은 0
```
