# 자리수 DP — 자릿수 합이 K인 수의 개수

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★★ 중 — 빈출 |
| 난이도 | 고급 |

## 함수 인터페이스

```ts
export function digitDp(N: number, K: number): number;
```

## 제약 조건

- $1 \leq N \leq 10^{15}$ (number 안전 정수 범위 내)
- $0 \leq K \leq 135$ (최대 자릿수 $15 \cdot 9 = 135$)

## 문제 상세

양의 정수 $N$ 과 $K$ 가 주어진다.

$1$ 이상 $N$ 이하의 정수 중, 십진법 자릿수의 합이 정확히 $K$ 인 수의 개수를 구한다.

## 예시

```ts
digitDp(20, 2);    // 2   (2, 11, 20 중 자릿수 합 2인 수는 2, 11, 20 → 3? 실제: 2, 11, 20 → 합 2: 2, 11, 20)
digitDp(100, 1);   // 3   (1, 10, 100)
digitDp(9, 5);     // 1   (5)
digitDp(99, 18);   // 1   (99)
```

| `N` | `K` | 반환값 |
| --- | --- | --- |
| `9` | `5` | `1` |
| `100` | `1` | `3` |
| `99` | `18` | `1` |
| `1000` | `3` | `19` |
