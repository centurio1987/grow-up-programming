# 외판원 문제 — 비트마스크 DP (TSP)

## 중요도 · 난이도

| 항목 | 값 |
|------|-----|
| 중요도 | ★ 하 — 특정 분야·고급 |
| 난이도 | 고급 |

## 함수 인터페이스

```ts
export function tspBitmask(dist: number[][]): number;
```

## 제약 조건

- $1 \leq n \leq 20$ (도시 개수, $dist$ 는 $n \times n$ 행렬)
- $0 \leq dist[i][j] \leq 10^6$
- $dist[i][i] = 0$

## 문제 상세

$n$ 개 도시의 거리 행렬 $dist[i][j]$ 가 주어진다.

도시 $0$ 에서 출발하여 **모든 도시를 정확히 한 번씩 방문**하고 다시 도시 $0$ 으로 돌아오는 **최소 비용**을 구해 반환하라.

$n = 1$ 인 경우 비용은 $0$ 이다.

## 예시

```ts
tspBitmask([[0]]);
// 0

tspBitmask([
  [0, 10],
  [10, 0],
]);
// 20   (0 → 1 → 0)

tspBitmask([
  [0, 10, 15, 20],
  [10, 0, 35, 25],
  [15, 35, 0, 30],
  [20, 25, 30, 0],
]);
// 80   (0 → 1 → 3 → 2 → 0: 10+25+30+15 = 80)
```
