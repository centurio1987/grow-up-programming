# 0/1 배낭 문제 (0/1 Knapsack)

## 함수 인터페이스

```ts
export function knapsack01(
  weights: number[],
  values: number[],
  W: number,
): number;
```

## 제약 조건

- $1 \leq n \leq 100$ (여기서 $n$ 은 `weights` (= `values`) 의 길이, 즉 물건의 개수)
- $1 \leq W \leq 10^4$ (배낭의 최대 용량)
- $1 \leq w_i, v_i \leq 10^4$ (각 물건의 무게와 가치)
- `weights` 와 `values` 의 길이는 같다

## 문제 상세

무게 $w_i$ 와 가치 $v_i$ 를 가진 $n$ 개의 물건이 있다.

배낭의 최대 적재 용량이 $W$ 일 때, 각 물건을 **0번 또는 1번만** 담아 가치 합을 최대로 하는 방법을 구한다. 그 최대 가치를 반환하라.

## 예시

```ts
knapsack01([1, 2, 3], [6, 10, 12], 5);    // 22   (인덱스 1, 2 선택: 무게 5, 가치 22)
knapsack01([2, 3, 4, 5], [3, 4, 5, 6], 5); // 7    (인덱스 0, 2 선택: 무게 6? 실제: 0+2=6>5 → 1,3 무게8 → 1만 무게3 가치4 +0 무게5 가치7)
knapsack01([5], [10], 4);                 // 0    (담을 수 없음)
knapsack01([1, 1, 1], [1, 2, 3], 2);      // 5    (인덱스 1, 2 선택)
```
